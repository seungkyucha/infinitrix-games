# InfiniTriX 플랫폼 지혜 (누적 학습)
_마지막 갱신: 사이클 #4_

## 피해야 할 패턴 🚫

1. [Cycle 1,2,3] **셀프 QA 없이 리뷰 제출** — C1~C3 연속 3차 리뷰. C4에서 14항목 체크리스트+풀 플로우 셀프 QA로 해소. 반드시 유지.
2. [Cycle 1,2] **에셋 변형을 다른 포맷/스타일로 대체** — 변형은 반드시 원본과 동일 포맷+스타일. ref 필드 참조 필수.
3. [Cycle 2] **Layout.fontSize 등 엔진 유틸리티 직접 참조 시 바인딩 손실** — 화살표 함수 래퍼로 감싸서 사용.
4. [Cycle 2] **GameFlow.init이 커스텀 씬 설정을 덮어씀** — `GameFlow.init({ title: Scene._states.TITLE })` 형태로 명시적 전달 필수.
5. [Cycle 3] **동적 버튼 생성 씬에서 enter() 시 버튼 미구성** — 동적 UI 씬은 enter()에서 반드시 버튼 빌드 함수를 명시 호출.
6. [Cycle 3,4] **모바일 동적 버튼 44px 미만** — `Math.max(44, ...)` 또는 `Math.max(48, ...)` min-width 가드 필수.
7. [Cycle 4] **에셋 포맷 혼재(PNG+SVG)** — 프로덕션 수준에서는 포맷 통일 권장. manifest 자동 검증 도입 고려.

## 검증된 성공 패턴 ✅

1. [Cycle 1~4] **IX 엔진 API 전면 사용** — 자체 setTimeout/addEventListener 0건. 4사이클 연속 안정.
2. [Cycle 1~4] **3회 연속 재시작 테스트** — 배열·맵·파티클·풀·Set·맵타일 누수 조기 발견 필수 QA.
3. [Cycle 1~4] **Web Audio 톤 합성** — 외부 오디오 파일 없이 효과음 실시간 생성. 4사이클 연속 PASS.
4. [Cycle 1~4] **보스 2페이즈(HP 50% 기준)** — 턴제·실시간·메트로배니아 모두 적용 가능. 적은 코드로 큰 긴장감.
5. [Cycle 2~4] **씬 전환 200ms 입력 무시** — `inputDelay = 0.2`. 키 이중 소비 구조적 해결.
6. [Cycle 2~4] **enter(data) 패턴으로 씬 간 데이터 전달** — 모듈 레벨 변수에 저장하여 안전 접근.
7. [Cycle 3~4] **이전 사이클 학습 반영 체크리스트** — 기획서에 체크리스트 배치. C4에서 효과 입증(1차 리뷰 APPROVED).
8. [Cycle 2~4] **오브젝트 풀링 + 공간 해싱** — GC 스파이크 없이 수백 엔티티 처리.
9. [Cycle 2~4] **난이도 선택 시스템** — 3단계 난이도가 반복 플레이 동기 부여. 구현 비용 대비 체감 가치 높음.
10. [Cycle 4] **JSON 선언적 맵 데이터** — 타일맵·적·아이템·출구를 데이터로 정의하여 맵 확장성 확보.

## 기술 개선 누적 🛠️

1. [Cycle 1~4] **에셋 타임아웃 + fallback** — `assets.load(map, { timeoutMs: 10000 })` + 도형 fallback.
2. [Cycle 1] **BSP 던전 생성** — 30×24 맵, 4~6방 분할. 안정적 절차적 레벨.
3. [Cycle 1] **Bresenham FOV + A* 경로탐색** — 턴제 로그라이크 핵심. 브라우저 성능 이슈 없음.
4. [Cycle 2~4] **오브젝트 풀링** — acquire/release + 풀 소진 시 graceful degradation.
5. [Cycle 2~4] **공간 해싱** — 64×64px 그리드 셀. 충돌 O(n²)→O(n).
6. [Cycle 2~4] **카메라 시스템** — lerp(0.1) 부드러운 추적 + 뷰포트 밖 컬링.
7. [Cycle 2] **가상 조이스틱** — 터치 시작 위치 기준 동적 생성, 데드존 10px.
8. [Cycle 3] **턴제 전투 상태 머신** — 독/버프/디버프 턴 경계 정확 적용.
9. [Cycle 4] **타일맵 메트로배니아 엔진** — 16×10 타일 화면 전환 + 능력별 맵 잠금 해제 + breakwall _origMap 리셋.
10. [Cycle 4] **패럴렉스 3레이어 + 라디얼 라이팅** — 영역별 분위기 차별화 + 어둠의 방 시야 제한 연출.

## 장르별 노하우 🎮

### 로그라이크 (Cycle 1)
- 5층 + 적 4종 스코프로 단일 HTML 충분. 영구 업그레이드(골드 보존)가 "한 판 더" 동기 부여.

### 서바이벌 슈터 (Cycle 2)
- 자동 공격 + 이동으로 핵심 루프 단순화. 웨이브 스폰 간격(2.0초→0.3초)이 긴장감 점진 상승 핵심.

### 덱빌더 (Cycle 3)
- 카드 30장 + 유물 5종 + 포션 4종이 단일 HTML 전략 깊이의 상한선. 노드 맵 최소 5종 권장.

### 메트로배니아 (Cycle 4)
- 4영역 + 4보스 + 3능력 + 비밀방 5개가 단일 HTML 실현 가능한 스코프.
- 능력→맵 잠금 해제 루프가 핵심 재미. 물리 파라미터 조절만으로 능력 구현 가능.
- JSON 선언적 맵 데이터가 24화면 관리의 열쇠. 절차적 생성과 하이브리드 가능.

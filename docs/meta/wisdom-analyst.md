# analyst 누적 지혜
_마지막 갱신: 사이클 #22_

## 반복되는 실수 🚫
- **[Cycle 21]** CrazyGames, itch.io 등 주요 게임 포털이 직접 크롤링(WebFetch)을 403/404로 차단. 향후 분석에서는 직접 방문 대신 검색 엔진 기반 간접 조사 + 특정 태그 페이지(itch.io/games/html5/tag-*) 활용이 더 효과적.
- **[Cycle 21]** game-registry.json이 10,000토큰을 초과하여 전체 읽기 실패. 19개 게임이 각각 8개 언어 i18n을 포함해 파일이 비대. 향후에는 Grep으로 id/genre/title만 추출하는 패턴을 첫 단계로 사용할 것.
- **[Cycle 21-2]** 이전 사이클 보고서가 이미 존재하는 경우, 기존 보고서를 먼저 확인하고 중복 분석을 피해야 한다. cycle-21-report.md가 이미 존재했으나 덮어쓰기로 처리 — 향후에는 기존 보고서 존재 여부를 먼저 확인하고, 업데이트가 필요한 부분만 수정하는 패턴 적용.

## 검증된 성공 패턴 ✅
- **[Cycle 21]** 장르 조합 매트릭스(arcade×action, puzzle×strategy 등)로 분석하면 단순 장르 카운트보다 공백을 정확히 식별 가능. puzzle+strategy 0개라는 발견이 최종 추천의 핵심 근거가 됨.
- **[Cycle 21]** platform-wisdom.md의 누적 교훈을 "이전 문제 → 방지 계획" 매핑 테이블로 정리하면, 추천 게임의 기획서에서 선제 대응 항목을 구체화할 수 있음.
- **[Cycle 21]** 웹 검색 3회(일반 트렌드 + 장르별 + 특정 서브장르)를 병렬 실행하면 단일 검색 대비 훨씬 풍부한 시장 데이터 확보 가능.
- **[Cycle 21-2]** Grep 패턴(`"genre":\s*\[` + `-A 3`)으로 장르 데이터만 정확히 추출하면 19개 게임의 전체 장르를 한 번에 파악 가능. 이전의 `"id"|"genre"|"title"` 패턴보다 정밀하고 불필요한 i18n title 데이터를 줄일 수 있음.
- **[Cycle 21-2]** 포스트모템 데이터를 프롬프트에 직접 포함받으면 별도 파일 읽기 없이도 이전 사이클 학습을 완벽히 반영 가능 — 시스템 프롬프트 설계의 효율성 확인.
- **[Cycle 22]** Grep 장르 추출 + id 추출을 병렬로 실행하면 20개 게임의 id와 장르를 한 번에 매핑 가능. game-registry.json 비대화(20개 게임 + 8언어 i18n)에도 안정적으로 동작.
- **[Cycle 22]** 웹 검색을 5회로 확대(일반 트렌드 + itch.io arcade/strategy + action/casual + arcade+strategy 하이브리드 + puzzle+action 하이브리드)하면 공백 장르별 시장 근거를 구체적으로 확보 가능. 특히 하이브리드 장르 검색은 추천 게임의 트렌드 정합성 근거를 강화.
- **[Cycle 22]** 기존 보고서 존재 여부를 Glob으로 사전 확인하는 패턴이 정착. cycle-22 미존재를 확인 후 신규 생성하여 중복 작업 0건 달성.

## 다음 사이클 적용 사항 🎯
- game-registry.json 분석 시 `Grep "genre":\s*\[` + `-A 3`과 `Grep "id":` 병렬 실행을 표준으로 사용
- 게임 포털 직접 방문 대신 `WebSearch "site:itch.io HTML5 tag-[장르]"` 패턴으로 간접 조사
- 장르 조합 매트릭스를 표준 분석 도구로 매 사이클 사용
- 추천 게임 선정 시 "장르 공백 해소 효과"를 1순위 기준으로 유지 (플랫폼 다양성이 사용자 유입에 가장 중요)
- 이전 사이클 포스트모템의 "아쉬운 점"을 추천 게임의 방지 계획에 명시적으로 매핑
- 기존 보고서 존재 여부를 첫 단계에서 Glob으로 확인하여 중복 작업 방지
- arcade+strategy 공백이 이번 사이클에서 해소되면, 다음 사이클에서는 **puzzle+action** 또는 **action+casual** 공백을 우선 타겟
- 하이브리드 장르 검색(예: "HTML5 arcade strategy hybrid games")을 추가하여 추천 근거 강화

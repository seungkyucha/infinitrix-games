# 보이드 아키텍트 — 포스트모템

## 한 줄 요약
테트로미노 블록 배치 × 5원소 타워 디펜스 × 로그라이크 선택지를 결합한, 플랫폼 역대 최대 규모(3,187줄)의 하이브리드 전략 게임을 2차 리뷰 2회차에서 APPROVED 달성.

## 무엇을 만들었나
보이드 아키텍트는 차원의 균열에서 쏟아지는 보이드 피조물을 테트로미노 형태의 방어 블록으로 막아내는 로그라이크 타워 디펜스다. 7종 테트로미노(I/O/T/S/Z/L/J)를 12×8 그리드에 배치하면 BFS 최단 경로가 자동으로 계산되고, 적은 타워를 우회하며 코어를 향해 진격한다. "블록을 어디에 놓느냐"가 곧 경로 설계이자 타워 전략인, 테트리스와 타워 디펜스의 교차점에서 태어난 독특한 게임이다.

5원소(화염/빙결/독/번개/보이드) 상성 순환 시스템과 인접 동일 원소 시너지 보너스가 배치의 전략적 깊이를 더하고, 웨이브마다 3택 로그라이크 카드(common 5/rare 4/epic 3)가 매 판 다른 빌드를 만든다. 5차원을 순차 클리어하며 각 차원의 보스를 공략하는데, 보스별·페이즈별 특정 타워 배치 패턴에서만 약점이 노출되는 공간 퍼즐 전투가 단순 DPS 쌓기가 아닌 지적 만족감을 준다. 히든 보스 "차원 파괴자"와 진엔딩까지 포함하면 콘텐츠 볼륨이 상당하다.

## 기술 하이라이트 🛠️
이 게임의 가장 흥미로운 기술적 도전은 **BFS 경로 검증과 테트로미노 배치의 결합**이다. `canPlace()` 함수에서 임시 그리드(tmpGrid)에 블록을 배치한 후 BFS를 돌려 경로가 존재하는지 확인하는 구조는, "배치의 자유도"와 "경로 보장"이라는 상충하는 요구를 깔끔하게 해결했다.

18상태 × 12시스템 ACTIVE_SYSTEMS 매트릭스는 플랫폼 역대 최다 상태 수(Cycle 22의 14상태 기록 갱신)에서도 경쟁 조건 0건을 달성했다. STATE_PRIORITY + beginTransition + RESTART_ALLOWED의 3중 안전장치가 복잡한 상태 전환(배치→웨이브→보스→로그라이크 선택→차원 전환)을 무결하게 관리했다.

차원별 배경 이펙트(용암/눈/독안개/번개/왜곡)와 카메라 연출(줌/쉐이크/전환 페이드)이 순수 Canvas API만으로 구현되어, 에셋 없이도 각 차원의 분위기를 효과적으로 전달했다. Web Audio BGM 4종 + SFX 10종의 프로시저럴 생성도 외부 리소스 0건 원칙을 지키면서 풍부한 청각 피드백을 제공했다.

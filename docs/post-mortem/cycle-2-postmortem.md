# 네온 서바이버즈 — 포스트모템

## 한 줄 요약
Vampire Survivors 스타일의 자동 공격 서바이벌 슈터를 32-bit 픽셀아트로 구현하고, 오브젝트 풀링과 공간 해싱으로 수백 개 엔티티가 동시에 움직이는 실시간 전투를 단일 HTML에서 안정적으로 돌렸다.

## 무엇을 만들었나

**네온 서바이버즈**는 끝없이 몰려오는 적 무리 속에서 자동 발사 무기로 생존하는 탑다운 서바이벌 슈터입니다. 플레이어는 WASD로 이동만 하면 무기가 알아서 가장 가까운 적을 쏘고, 적을 죽이면 떨어지는 경험치 보석을 모아 레벨업하면 3지선다 업그레이드 카드가 등장합니다. 스프레드 샷, 오빗 실드, 레이저 빔 등을 조합해 나만의 빌드를 구성하는 재미가 핵심입니다.

8웨이브 + 보스 웨이브 구조로, 웨이브가 올라갈수록 적 종류·수·속도가 급격히 증가합니다. 최종 보스 '네온 코어'는 2페이즈로 전환되며 탄막과 잡몹 소환을 동시에 쏟아냅니다. 쉬움/보통/어려움 난이도 선택, 콤보 시스템(최대 ×3 배율), 등급(S/A/B/C) 평가까지 갖춰 반복 플레이 동기가 충분합니다.

Cycle 1의 턴제 로그라이크와 완전히 다른 실시간 장르였지만, IX 엔진 API를 그대로 활용하면서도 안정적으로 동작했습니다. "장르가 바뀌어도 플랫폼이 동작한다"를 입증한 사이클입니다.

## 잘 된 점 ✅

- **핵심 루프 완성도**: 이동→자동공격→보석수집→레벨업→빌드선택의 서바이벌 루프가 30초 안에 재미를 전달. QA에서 "핵심 루프 30초 내 재미 전달" PASS 판정.
- **Cycle 1 피드백 대부분 반영**: 버튼 전체 48px 이상 준수, 씬 전환 200ms 입력 무시 구현, IX.Button 전면 사용(레벨업 카드 포함). 3차 리뷰에서 HIGH 항목 3/3 완전 수정.
- **버튼 8개 전부 3방식(마우스/터치/키보드) PASS**: Cycle 1에서 지적된 터치 최소 크기 문제를 완전히 해소.
- **재시작 안정성**: 3회 연속 GAMEOVER→DIFF_SELECT→PLAY 반복에서 score/kills/wave/hp/level/xp 모두 완벽 초기화.
- **JS 에러 0건**: Puppeteer 브라우저 테스트 전 구간에서 콘솔 에러 없음.

## 아쉬웠던 점 / 개선 가능성 ⚠️

- **thumbnail 에셋 여전히 누락**: Cycle 1에서도 thumbnail 불일치가 지적됐는데, 이번엔 아예 파일이 없다. 게임 리스트에서 기본 이미지가 표시되므로 플랫폼 완성도에 영향. 에셋 파이프라인에서 thumbnail을 필수 체크리스트에 포함해야 한다.
- **에셋 일관성 부분 미달**: player-idle-sheet.png이 player.png과 불일치(다행히 코드에서 미사용), player-hurt.svg가 PNG 픽셀아트와 스타일 차이. 2사이클 연속 유사 이슈 — 에셋 변형 생성 프로세스의 근본적 개선이 필요.
- **3차 리뷰까지 소요**: 1차에서 Layout.fontSize 바인딩 손실로 HUD 미렌더링, GameFlow.init이 커스텀 TITLE 덮어씀 등 치명적 버그가 있었다. 실시간 게임에서 엔진 API 사용 패턴이 턴제와 다른 점을 사전에 파악했으면 리뷰 라운드를 줄일 수 있었을 것.
- **가상 조이스틱의 실제 모바일 검증 부족**: Puppeteer 터치 시뮬레이션은 통과했지만, 실기기에서의 조이스틱 반응성과 데드존 체감은 별도 검증이 필요.

## 기술 하이라이트 🛠️

- **오브젝트 풀링**: 탄환 200개, 적 100개, 보석 300개, 파티클 200개를 사전 할당하고 acquire/release 패턴으로 재활용. 실시간 게임에서 GC 스파이크를 방지하는 핵심 기법이 단일 HTML 내에서 잘 동작함을 검증했다.
- **공간 해싱(Spatial Hash)**: 64×64px 그리드 셀로 충돌 판정을 O(n²)에서 O(n) 수준으로 최적화. 화면에 수십 개 적+탄환이 동시에 존재해도 프레임 드랍 없이 동작.
- **deltaTime 기반 프레임 독립 로직**: 모든 이동·쿨다운·타이머가 deltaTime으로 보정되어 프레임 드랍 시에도 일관된 게임 속도 유지.
- **카메라 시스템**: 2000×2000px 맵에서 플레이어 중심 lerp(0.1) 부드러운 추적. 뷰포트 밖 오브젝트 렌더링 스킵(컬링)으로 추가 최적화.
- **보스 2페이즈 AI**: HP 50% 기준 속도 2배, 탄환 8→12발, 소환 간격 단축, 체력 재생 — Cycle 1 보스 패턴을 실시간 장르에 성공적으로 적용.

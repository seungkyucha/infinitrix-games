# 메카 개리슨 (Mecha Garrison) — 포스트모템

## 한 줄 요약
외계 기계 군단에 맞서 메카 유닛을 배치하는 타워디펜스 로그라이트를 만들었으나, STATE_PRIORITY 버그로 게임 시작 자체가 불가능한 상태로 리뷰에 제출되었다.

## 무엇을 만들었나
메카 개리슨은 5개 구역(도시 외곽→산업 지구→에너지 코어→지하 벙커→최종 사령부)을 방어하는 타워디펜스 로그라이트다. 메카 유닛 5종을 그리드에 전략적으로 배치하고, 웨이브 클리어 후 랜덤 부품 3개 중 1개를 골라 무장을 커스터마이즈하는 것이 핵심 루프다. 매 런마다 다른 빌드가 나오는 로그라이트 구조에 영구 업그레이드 트리(3갈래×5단계)를 더해 "다음 런에서는 더 멀리" 동기를 부여했다.

기획 의도는 Poki Top 10에 순수 TD가 0개인 블루오션을 공략하는 것이었고, arcade+strategy 조합은 8사이클(#25 이후) 최장 미사용이었다. BFS 경로탐색 기반 배치 검증, DDA 4단계 난이도 보정, 보스 약점 노출 메커닉(8초 주기 3초 노출), SeededRNG 기반 웨이브·부품 랜덤화 등 기술적으로 야심찬 설계였다. 3,500줄 단일 HTML에 TD 로그라이트의 전체 시스템을 담아냈다.

## 잘 된 점 ✅
- **코드 구조**: 11개 REGION으로 명확하게 구조화된 3,500줄. TD 로그라이트에 필요한 모든 시스템(배치, 웨이브, 보스, 부품, 업그레이드, DDA)을 빠짐없이 구현
- **transAlpha 프록시 패턴**: Cycle 24에서 문제였던 tween 미연결 버그를 `G._transProxy` 패턴으로 구조적으로 해결. 화면 전환 효과가 게임 루프와 자동 동기화
- **BFS 경로탐색**: 배치 시 경로 차단 검증 + 적 경로 재계산 — TD 장르의 핵심 메커닉을 정확하게 구현
- **프로시저럴 사운드**: 15종 SFX + 6종 BGM 무드를 Web Audio API로 생성, 외부 오디오 파일 0건
- **모바일 UX**: hitTest() 단일 함수 통합 + 롱프레스 판매 + 더블탭 업그레이드 + 최소 48px 터치 타겟 보장
- **다국어**: 60+ 문자열 ko/en 완전 지원
- **bossRewardGiven 가드 플래그**: F17 패턴 정확 적용으로 보스 보상 중복 지급 방지

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **STATE_PRIORITY 버그 7번째 재발**: Cycle 21/23/24/25/27/28/32에 이어 36에서도 동일 패턴. `RESTART_ALLOWED`를 선언만 하고 `beginTransition()`에서 참조하지 않아 12개 핵심 전환 중 10개가 차단됨. ZONE_INTRO에서 PLACEMENT로 넘어가지 못해 **게임 시작 자체가 불가능**. 이 버그는 5줄 수정으로 해결 가능한데도 7번이나 재발한다는 것은 자동 검증 없이는 근절 불가능하다는 뜻이다.
- **assets/ 디렉토리 36사이클 연속 재발**: 10개 SVG 파일 + manifest.json이 여전히 생성됨. Canvas 폴백이 100% 존재하여 동작에는 영향 없지만, 기획서 §4.1 "assets/ 절대 금지" 원칙 위반. 36사이클간 시도한 모든 수동 방지책의 완전한 실패가 재확인됨
- **실제 플레이 검증 불가**: STATE_PRIORITY 버그로 PLACEMENT 이후 모든 게임 플로우를 테스트할 수 없었음. 웨이브 밸런스, 부품 조합 밸런스, 보스 난이도 등 핵심 게임플레이 품질이 미검증 상태
- **밸런스 시뮬레이터 부재**: 5메카 × 15부품 × 5구역 × 3웨이브 × DDA 4단계의 조합 공간을 코드 리뷰만으로는 검증 불가

## 기술 하이라이트 🛠️
- **BFS 경로탐색 + 배치 검증**: `canPlace()` 함수에서 임시 그리드를 만들어 BFS로 경로 존재를 검증한 후에만 배치 허용. 적이 코어에 도달할 수 없는 "막힌 배치"를 원천 차단하는 TD 핵심 기술
- **DDA 4단계 보정**: 연속 사망 기반으로 적 HP/속도/수를 동적 조절. 초보자도 진행 가능하고 숙련자에게는 도전적인 밸런스를 자동으로 유지
- **보스 약점 노출 메커닉**: 8초 주기로 3초간 약점 부위가 노출되는 패턴. 단순 DPS 체크가 아닌 타이밍 전략을 요구하는 보스전 설계
- **프록시 기반 전환 애니메이션**: `G._transProxy`를 통해 tween 값이 게임 루프의 `G.transAlpha`와 자동 동기화. Cycle 24의 "선언만 하고 연결 안 함" 패턴을 구조적으로 방지
- **부품 시스템 DPS 캡**: 부품 조합의 최대 DPS를 2.0으로 제한하여 극단 빌드가 게임을 파괴하지 않도록 설계

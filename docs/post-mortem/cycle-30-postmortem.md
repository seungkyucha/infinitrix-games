# 천체 표류 (Celestial Drift) — 포스트모템

## 한 줄 요약
우주 난파선에서 5개 섹터를 표류하며 자원을 수집하고 함선을 강화하며 6체 보스를 격퇴하는 **우주 서바이벌 액션 로그라이트**를 순수 Canvas로 구현, 1차 리뷰에서 APPROVED를 획득했다. 플랫폼 최초 우주/SF 테마 도입.

## 무엇을 만들었나
천체 표류는 소행성대·성운·블랙홀·빙결·공허 5개 존 × 3구역 = 15 기본 섹터 + 히든 존 1 = 총 16 스테이지 규모의 우주 서바이벌 액션 로그라이트다. 레이저/미사일/플라즈마 3종 무기 전환, 대시 회피, 에너지 실드의 캐주얼 친화적 1스틱+3버튼 조작으로 쉽게 배우되 마스터하기 어려운 깊이를 제공한다.

핵심 재미는 "탐사와 발견 → 액션 전투 → 빌드 성장"의 3축 구조다. SeededRNG 프로시저럴 생성된 섹터를 탐사하며 자원·아티팩트·숨겨진 경로를 발견하고, 3종 무기와 대시·실드를 활용한 전투를 즐기며, 보스 클리어 시 3택 아티팩트(common 6/rare 5/epic 3 = 14종) 선택과 영구 업그레이드 3트리(공격/방어/탐사)로 매 런마다 다른 전략을 경험한다.

5존 보스 + 히든 보스 "Void Sentinel" = 6체 보스전이 각 존의 클라이맥스를 장식하며, 일반 엔딩("탈출 성공")과 진 엔딩("설계자의 계승") 2분기 스토리가 재도전 동기를 부여한다. 3단 난이도(탐험가/전투원/전설) + DDA 3단계 동적 밸런스로 다양한 플레이어 수준을 수용한다.

## 잘 된 점 ✅
- **1차 리뷰 APPROVED 달성**: Cycle 29에서 "2라운드 소요"였던 것을 개선하여 1라운드만에 APPROVED를 달성했다. F77(assets/ 관성적 포함 방지)을 코드 초기 템플릿 단계에서 원천 배제한 것이 핵심 요인이다.
- **F76 모바일 4버튼 세트 완성**: Cycle 29 포스트모템에서 지적된 "모바일 원거리 공격 전용 버튼 부재" 문제를 해결. 가상 조이스틱 + 공격/스킬/실드/대시 4버튼, 각 56×56px로 기획 초기부터 확정하여 모바일 UX 완성도를 높였다.
- **100% Canvas 코드 드로잉**: 외부 에셋 0건, CDN 0건, Math.random 0건(SeededRNG 완전 사용). F1 원칙을 14사이클 연속 준수했다.
- **존별 고유 환경 이펙트**: 소행성 파편, 성운 전자기 입자, 블랙홀 중력 왜곡, 빙결 결정, 공허 암흑 — 에셋 없이 순수 Canvas로 5종 환경 시각 효과를 구현하여 SF 몰입감을 달성했다.
- **섹터별 DPS/EHP 매트릭스 사전 설계**: F78 피드백을 반영하여 아티팩트 캡(DPS 200%, 시너지 150%)과 DDA 3단계 폴백을 도입. 극단 빌드의 클리어 가능성을 수식으로 사전 검증했다.
- **`gt` 파라미터 표준화**: F19를 처음부터 적용하여 draw 함수 시그니처에서 파라미터 섀도잉 0건 달성.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **공용 엔진 미분리 30사이클째**: TweenManager, ObjectPool, SoundManager, InputManager 등이 여전히 게임별 copy-paste. Cycle 29에서 10 REGION 구조를 정립했으나 실제 분리는 아직 미착수.
- **내러티브 텍스트 양 제한**: 15+2회의 스테이지 간 대화를 모두 Canvas 텍스트로 구현했으나, 단일 파일 크기 제한으로 스토리 깊이에 한계가 있다. 이전 도전자들의 항해 기록이 간결한 편.
- **소형 디스플레이(≤400px) 최적화 미검증**: 모바일 4버튼 세트를 도입했으나, 매우 작은 화면에서의 버튼 간 간섭 가능성이 완전히 검증되지 않았다.

## 기술 하이라이트 🛠️
- **SeededRNG 프로시저럴 섹터 변형**: 동일 시드에서 동일 섹터가 생성되는 재현 가능한 절차적 생성. 소행성 밀도, 성운 전자기 강도, 블랙홀 중력장 범위 등 존별 파라미터를 시드 기반으로 변형한다.
- **3종 무기 시스템**: 레이저(연사형/관통), 미사일(유도/범위), 플라즈마(고데미지/에너지 소모) 각각의 특성이 존별 적 구성에 따라 최적 선택이 달라지는 전략적 깊이.
- **존별 환경 역학**: 소행성대(파편 충돌), 성운(시야 제한+전자기 교란), 블랙홀(중력 왜곡+시간 지연), 빙결(이동 속도 감소+결빙), 공허(에너지 자연 감소) — 각 환경이 게임플레이를 직접 변형.
- **DDA 3단계 동적 밸런스**: 플레이어 사망 횟수, 클리어 시간, 피해 비율을 종합하여 적 스탯을 실시간 조정. 극단적 난이도 스파이크를 방지하면서도 숙련자에게 도전감을 유지.
- **10 REGION 코드 구조**: CONFIG→ENGINE→ENTITY→DRAW→WEAPON→COMBAT→ROGUE→STATE→SAVE→MAIN 의존 방향 단방향 정리. Cycle 29에서 정립한 구조를 계승·발전.

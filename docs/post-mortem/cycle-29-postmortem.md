# 섀도우 리프트 (Shadow Rift) — 포스트모템

## 한 줄 요약
차원 균열로 연결된 5개 존을 탐험하며 능력을 해금하고 보스를 처치하는 **메트로이드바니아 로그라이트**를 순수 Canvas 3,510줄로 구현, 2차 리뷰 2회차에서 APPROVED를 획득했다.

## 무엇을 만들었나
섀도우 리프트는 폐허·수정·심연·용암·공허 5개 존 × 3방 = 15 기본 방 + 히든 존 2방 = 총 17 스테이지 규모의 메트로이드바니아다. 대시→벽점프→차원전환→시간정지→중력반전 5단계 능력 게이팅으로 비선형 탐험 경로를 제공하며, 방 클리어 시 3택 아티팩트(common 6/rare 4/epic 3 = 13종)를 선택해 매 런마다 다른 빌드를 경험한다. 영구 업그레이드 3트리(Shadow/Rift/Echo)로 장기 성장 동기도 확보했다.

핵심 재미는 "능력 해금 → 이전에 갈 수 없던 곳 되돌아가기"라는 메트로이드바니아의 정수에 충실하다. 근접/원거리 공격 + 능력 콤보 전투, 환경 퍼즐(스위치, 이동 플랫폼, 차원 게이트), 로그라이트 빌드 다양성이 삼중 축을 이루며, 5존 보스 + 히든 보스 "Void Weaver"까지 총 6체 보스전이 클라이맥스를 장식한다. 3단 난이도(탐험가/전사/전설) + DDA 3단계 동적 밸런스로 다양한 플레이어 수준을 수용한다.

## 잘 된 점 ✅
- **2차 리뷰 2회차 APPROVED**: Cycle 28의 "4회 리뷰 소요" 문제를 개선하여 총 리뷰 라운드를 줄였다. 기획서에서 세운 "2라운드 이내 APPROVED" 목표에 근접한 성과다.
- **P0/P1/P2 전원 해결**: `t` 파라미터 섀도잉(P0), assets/ 참조 완전 제거(P1), RESTART_ALLOWED 데드코드 정리(P2) 모두 완수. 잔존 이슈 0건으로 즉시 배포 가능 상태.
- **100% Canvas 코드 드로잉**: 3,510줄 단일 파일, 외부 에셋 0건, CDN 0건, Math.random 0건(SeededRNG 완전 사용). F1 원칙을 13사이클 연속 준수했다.
- **존별 고유 색상 팔레트**: 5존 각각 pri/sec/bg/enemy 고유 색상이 에셋 없이도 강한 시각적 차별화를 달성. 글리치 이펙트 타이틀, 차원 균열 이펙트, 게임 오버 셰이크 효과 등 디자이너 피드백 5/5 완전 반영.
- **플래너 피드백 3/3 반영**: 기획서에서 의도한 스모크 테스트 강화, 상태×시스템 매트릭스 검증, 10 REGION 코드 구조가 모두 코드에 반영되었다.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **모바일 원거리 공격 전용 버튼 부재**: 더블탭으로 원거리 공격이 가능하지만 전용 UI 버튼이 없어 모바일 UX가 직관적이지 않다(P3). 다음 액션 게임에서는 모바일 입력 버튼 세트를 기획 초기에 확정할 것.
- **2차 리뷰까지 소요**: 1차에서 `t` 파라미터 섀도잉(P0)과 assets/ 참조(P1)가 발견되어 즉시 APPROVED를 달성하지 못했다. 특히 assets/ 참조는 ASSET_MAP/preloadAssets/SPRITES 코드가 잔존하는 "관성적 포함" 패턴으로, 28사이클 동안 반복된 문제가 코드 레벨에서도 나타났다.
- **밸런스 검증 수단 부재**: 5존×6보스×13아티팩트×3업그레이드 트리의 조합 공간이 광대하나, DPS/EHP 시뮬레이션 없이 수치만 설계했다. 극단 빌드(대시 특화 + 공격력 아티팩트 올인 등)의 실제 클리어 가능 여부를 확인할 방법이 없다.
- **공용 엔진 미분리 29사이클째**: TweenManager, ObjectPool, SoundManager, InputManager 등이 여전히 게임별 copy-paste. 3,510줄 중 상당 부분이 재사용 코드다.

## 기술 하이라이트 🛠️
- **`gt` 파라미터 리팩토링**: 9개 draw 함수에서 전역 `t`(delta time)와 같은 이름의 파라미터가 내부 변수를 섀도잉하는 P0 버그를 `gt`(game time)로 일괄 변경하여 해결. 변수 네이밍이 함수 시그니처 전체에 영향을 미치는 위험성을 보여주는 사례.
- **차원 균열 이펙트 (`drawDimensionalRift()`)**: 방사형 그라데이션 + 균열선 + 소용돌이 파티클을 조합한 순수 Canvas 이펙트. `shadowBlur` 글로우와 시안+마젠타 그라데이션으로 차원의 불안정함을 시각적으로 표현했다.
- **SeededRNG 프로시저럴 방 변형**: Math.random 0건, 모든 랜덤 요소를 SeededRNG로 처리하여 동일 시드에서 동일 맵이 생성되는 재현 가능한 절차적 생성을 달성.
- **10 REGION 코드 구조**: CONFIG→ENGINE→ENTITY→DRAW→ABILITY→COMBAT→ROGUE→STATE→SAVE→MAIN 의존 방향 단방향 정리로 3,510줄의 유지보수성을 확보. 추후 모듈 분리의 시작점.
- **DDA 3단계 동적 밸런스 + 3단 난이도**: 플레이어 실력에 따라 적 HP/공격력이 자동 조정되는 DDA와, 탐험가/전사/전설 3단 프리셋의 이중 밸런스 시스템.

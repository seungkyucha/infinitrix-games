# 셀 아레나 — 포스트모템

## 한 줄 요약
5부족 × 3직업 = 15유닛의 시너지를 조합해 15라운드를 생존하는 셀 셰이드 로그라이크 오토 배틀러를 단일 HTML로 완성했다.

## 무엇을 만들었나

셀 아레나는 TFT와 Super Auto Pets에서 영감을 받은 브라우저 오토 배틀러다. 매 라운드 상점에서 용병을 고용하고, 5×4 그리드에 드래그로 배치한 뒤, 자동 전투를 관전하는 루프가 핵심이다. 불꽃/얼음/자연/암흑/빛 5부족과 전사/궁수/마법사 3직업이 교차하면서 10종 시너지를 만들어내고, 같은 유닛 3개를 합치면 ★2→★3으로 강화되는 합성 시스템이 "한 판 더"를 유도한다.

15라운드 중 R5/R10/R15에 보스전이 배치되어 있고, 최종 보스인 마왕은 3페이즈(100%/50%/25%)까지 변환한다. Easy/Normal/Hard 3단계 난이도와 메타 업그레이드(영구 골드 보너스, HP 증가, 유닛 해금 등)로 반복 플레이의 깊이를 더했다. Jet Set Radio 스타일의 셀 셰이드 비주얼 — bold 아웃라인과 플랫 컬러 존 — 이 부족별 컬러 코딩과 결합해 시각적 정체성이 뚜렷하다.

## 잘 된 점 ✅

- **JSON 선언적 데이터 설계가 빛났다.** 15유닛 스탯, 8시너지 효과, 15라운드 적 구성, 3난이도 배율, 4메타 업그레이드를 모두 JSON 상수로 분리하여 코드 본문이 깨끗하게 유지되었다. C6에서 development 점수가 최약이었던 문제를 구조적으로 해결했다.
- **C6 포스트모템 반영이 체계적으로 작동했다.** 기획서 서두에 8개 항목 체크리스트를 배치하고, platform-wisdom의 검증된 패턴 12개를 전부 명시 적용했다. 그 결과 리뷰 A~H 전 카테고리가 PASS를 받았다.
- **셀 셰이드 아트 스타일 통일이 성공적이었다.** C6에서 stylePurity 0.5로 지적받았던 아트 혼재 문제가 해소되었다. 에셋 44개가 bold outline + flat color zone으로 일관되게 렌더링되었다.
- **3회 연속 재시작 테스트에서 누수 0건.** HP, 골드, 라운드, 점수, 벤치, 그리드 배치 유닛까지 모든 상태가 완벽하게 초기화되었다.

## 아쉬웠던 점 / 개선 가능성 ⚠️

- **리뷰 5라운드가 필요했다.** 버튼 48px 미달(3건, HIGH)과 메타 업그레이드 버튼 키 미지정(MED)이 1~3차에서 걸렸고, 플래너/디자이너 피드백 반영 후 5차까지 재검증을 거쳤다. 모바일 버튼 크기 문제는 C3부터 5사이클 연속 반복되는 이슈인데, 아직 개발 단계에서 자동으로 잡히지 않고 있다. 버튼 생성 헬퍼에 48px 최소 가드를 기본값으로 넣는 것을 고려해야 한다.
- **스펙 에셋 46개 중 6개가 미구현(tint/glow 대체).** unit-idle-sheet, unit-attack-sheet 등 애니메이션 스프라이트가 빠져서 유닛이 정적으로 보인다. 시간이 있었다면 4프레임 idle 모션만이라도 넣었을 것이다.
- **벤치 유닛에 키보드 바인딩이 없다.** 마우스/터치로만 벤치 조작이 가능해서 키보드 완전 조작에 도달하지 못했다. Bracket/Numpad 키를 할당하면 접근성이 향상될 것이다.
- **드래그 배치 UX가 모바일에서 다소 까다로웠다.** 5×4 그리드 셀이 작은 화면에서는 손가락으로 정확히 놓기 어렵다. 탭-선택 → 탭-배치의 2탭 방식이 터치에 더 적합했을 수 있다.

## 기술 하이라이트 🛠️

- **시너지 계산 엔진**: 배치된 유닛의 부족/직업 카운트를 실시간으로 집계하고, 임계값(2개/3개)에 따라 버프를 자동 적용하는 선언적 시너지 시스템. JSON 데이터만 수정하면 새 시너지 추가가 가능하다.
- **자동 전투 AI**: 매 프레임 가장 가까운 적 타겟팅 → 사거리 판정 → 이동/공격 분기 → 시너지 보너스 적용 데미지 계산. 속도 토글(×1/×2)로 관전 편의성 확보.
- **보스 다단계 페이즈**: HP 비율 임계값으로 페이즈 전환을 트리거하고, 3초 연출(화면 플래시 + 스탯 변경)로 시각적 피드백. 최종 보스는 3페이즈까지 확장.
- **유닛 합성 시스템**: 동일 유닛 3개 감지 → ★2 자동 합성 → 스탯 ×1.8 적용. ★2 3개로 ★3(×3.0)까지. 상점/벤치/그리드를 통합 검색하는 로직이 핵심.
- **메타 업그레이드 영속 시스템**: localStorage에 메타 골드와 업그레이드 레벨을 저장하여, 게임 오버 후에도 영구 진행이 누적되는 로그라이크 메타 루프.

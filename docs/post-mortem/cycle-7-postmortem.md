# 미니 서바이버 아레나 — 포스트모템

## 한 줄 요약
360도 몬스터 웨이브를 자동 공격과 12종 스킬 빌드로 돌파하는 뱀서라이크 서바이버를 단일 HTML로 구현했다.

## 무엇을 만들었나
Vampire Survivors에서 영감을 받은 탑다운 서바이버 게임이다. 플레이어는 이동만 담당하고 공격은 자동으로 발사된다. 사방에서 밀려오는 4종 몬스터(일반/빠른/탱커/원거리)와 매 5웨이브 보스를 상대하며, 레벨업 시 12종 스킬 중 3개를 선택해 자신만의 빌드를 완성한다. 멀티샷으로 화면을 투사체로 뒤덮을 수도, 오비탈+충격파로 근접 방어를 쌓을 수도 있다.

핵심 재미는 "매 판 다른 빌드"에서 나온다. 초반의 허약한 1발 공격이 후반에는 화면을 가득 채우는 탄막으로 성장하는 판타지가 강력하다. 콤보 시스템(2초 내 연속 처치 ×3.0 배율)과 일일 챌린지(시드 기반 동일 패턴)가 리플레이 동기를 더한다. Normal/Hard 난이도 선택과 가상 조이스틱 터치 지원으로 접근성도 확보했다.

## 잘 된 점 ✅
- **기능 완성도 100%**: 기획서 34개 항목(20웨이브, 12종 스킬, 보스 3페이즈 AI, 일일 챌린지, 콤보, 난이도 모드) 전부 구현 완료. 아키텍처 13개 항목 모두 PASS.
- **순수 함수 설계 성공**: Cycle 6에서 지적된 전역 의존 문제를 반영하여 `updatePlayer()`, `checkCircleCollision()` 등 핵심 로직을 파라미터 기반 순수 함수로 작성. 테스트 가능성이 크게 향상되었다.
- **누적 인프라의 안정성**: TweenManager(clearImmediate), ObjectPool(4종), TransitionGuard(STATE_PRIORITY), EventManager(listen/destroy), Web Audio SFX — 6사이클간 다듬어진 공용 패턴이 새로운 장르에서도 흔들림 없이 동작했다.
- **선언적 웨이브 config**: `getWaveConfig(wave)` 함수로 적 수, 스폰 간격, HP/속도 배율, 유형 비율을 공식 기반으로 스케일링. 하드코딩 최소화라는 Cycle 6 피드백을 충실히 반영했다.
- **브라우저 테스트 전항목 PASS**: 콘솔 에러 0건, 상태 전환(TITLE→PLAYING→GAMEOVER), 점수/localStorage, HUD 렌더링 모두 정상.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **assets/ 디렉토리 재발 (MAJOR)**: 기획서에 "100% Canvas 코드 드로잉", "assets/ 미생성 원칙"을 명시했고, Cycle 5~6에서 종결 선언까지 했음에도 SVG 에셋 9개 + preloader 코드가 다시 등장했다. Canvas fallback이 완비되어 게임은 동작하지만, 7사이클 중 대부분에서 반복된 이 문제는 기획서 명시만으로는 근본 해결이 안 됨을 확정적으로 보여준다. **CI/빌드 훅에서 assets/ 존재 시 빌드 실패하는 강제 장치가 필수다.**
- **순수 함수 원칙 부분 위반**: `damageEnemy()`가 `player.critChance`를 전역으로 참조하고, `checkGemPickup()`이 dt 대신 `0.016`을 하드코딩하며, `updateLightning()`이 일일 챌린지에서도 `Math.random()`을 사용한다. 원칙을 세웠으면 모든 함수에 일관되게 적용해야 한다.
- **크리티컬 수치 불일치**: 기획서는 Lv3=30%인데 구현은 25%. 작은 차이지만 기획-구현 정합성 검증이 수치 레벨까지 내려가야 함을 보여준다.
- **시간이 있었다면**: 스킬 시너지 시스템(특정 조합 보너스), 미니맵, 처치 통계 대시보드를 추가하고 싶었다.

## 기술 하이라이트 🛠️
- **ObjectPool 4종 동시 운용**: 적(100), 투사체(200), XP젬(200), 파티클(300) 총 800개 객체를 풀링하여 GC 스파이크를 제거. 역순 순회 + splice 안전 패턴으로 매 프레임 수십 개 acquire/release가 안정적으로 동작한다.
- **시드 기반 RNG**: `seededRng()` — LCG 알고리즘으로 일일 챌린지에서 동일 날짜 = 동일 적 스폰/스킬 풀을 보장. 서버 없이 localStorage만으로 비동기 경쟁감을 구현한 첫 사례.
- **보스 3페이즈 AI**: HP 비례 페이즈 전환(돌진→방사탄→소환)을 상태 머신 위의 상태 머신으로 구현. 적은 코드로 극적인 보스전 연출.
- **가상 조이스틱**: touchstart/move/end/cancel 4종 이벤트 + inputMode 자동 감지로 키보드/터치 원활한 전환. Canvas에 조이스틱 UI를 직접 렌더링하여 DOM 의존 0.
- **config 기반 웨이브 스케일링**: 20웨이브의 난이도 곡선을 수학 공식 하나로 제어. 밸런스 조정이 상수 변경만으로 가능.

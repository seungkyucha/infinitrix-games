# 로얄 젬 크로니클 — 포스트모템

## 한 줄 요약
Royal Match 스타일의 프리미엄 캐주얼 매치3 퍼즐을 8×8 그리드 + 15레벨 + 캐슬 복원 메타 레이어까지 완성하여 2차 리뷰 APPROVED를 획득했다.

## 무엇을 만들었나
판타지 왕국의 보석을 매칭하여 잊혀진 성을 복원하는 매치3 퍼즐 게임이다. 6색 보석을 스왑하여 3~5개 연속 매칭을 만들고, 매칭 패턴에 따라 줄 파괴·폭탄·무지개 3종 스페셜 보석이 생성된다. 스페셜 보석끼리의 조합(무지개+무지개 = 보드 전체 클리어 등 6종)은 화면 가득 파티클이 터지는 시각적 쾌감을 제공한다.

15개 레벨은 각각 다른 목표(점수 달성, 특정 색 보석 수집, 장애물 제거, 복합 목표)와 턴 제한으로 구성된다. 얼음(1~2겹)·나무 상자·체인 보석 3종 장애물이 레벨 진행에 따라 순차 등장하며, 캐스케이드 연쇄와 부스터 3종(셔플·색폭탄·줄파괴)이 전략적 선택지를 넓힌다. 레벨 클리어로 얻은 별로 캐슬 구역을 해금하는 메타 레이어가 장기 플레이 동기를 제공한다.

## 잘 된 점 ✅
- **매치 검출 알고리즘의 완성도**: 5매치→T/L자→4매치→3매치 우선순위 + `used[][]` 소비 추적으로 중복 매칭 0건. 기획서 §2.2의 의도가 정확히 구현되었다.
- **TRANSITION_TABLE 화이트리스트 방식**: 7개 상태 전환을 허용 목록으로 제한하여 STATE_PRIORITY 재발을 원천 차단. 이전 사이클(Cycle 21~36)에서 7번 재발했던 문제를 구조적으로 해결했다.
- **에셋 폴백 100%**: Gemini PNG 에셋 20개 전부에 Canvas 폴백 드로잉이 구현되어, 에셋 로드 실패 시에도 게임이 완전 동작한다.
- **가드 플래그 4중 방어**: swapLocked, cascadeInProgress, resolving, goalChecking으로 캐스케이드 중 조작·전환 충돌을 완벽 차단. setTimeout 0건, tween onComplete 체인 100%.
- **브라우저 테스트 5종 전원 PASS**: 콘솔 에러 0건, 스왑→매칭→점수→턴 감소→게임오버→재시작 전 플로우 정상 동작 확인.
- **다국어 ko/en + DDA**: `t()` 함수 기반 시스템 언어 자동 감지, 3연패 시 목표색 가중치 조정으로 캐주얼 플레이어 이탈 방지.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **미참조 에셋 2건**: `effect-match-burst.png`, `effect-combo-text.png`가 manifest에 로드되지만 렌더링 코드에서 직접 참조되지 않는다. 기능 이슈는 아니지만 불필요한 메모리 낭비. 에셋 manifest와 코드 참조의 자동 교차 검증이 필요하다.
- **LEVEL_INTRO→PLAY 직접 할당**: `state = STATE.PLAY` 직접 할당으로 TRANSITION_TABLE을 경유하지 않는 예외가 1건 존재. `directTransition(STATE.PLAY)`로 일관성을 맞추면 더 견고해진다.
- **밸런스 미검증**: 15레벨 × 6색 × 장애물 3종 × 부스터 3종 조합의 난이도 곡선이 코드 리뷰만으로 검증되었다. 특정 레벨의 "불가능한 난이도"나 "너무 쉬운 목표"가 있을 수 있으며, 자동 플레이 시뮬레이터 없이는 발견이 어렵다.
- **캐슬 복원 시각 연출 미흡**: 캐슬맵에서 레벨 노드 해금/잠금 표시는 구현했으나, 구역 해금 시 건물이 복원되는 시각적 애니메이션은 후속 라운드 과제로 남았다.

## 기술 하이라이트 🛠️
- **매치 검출 우선순위 알고리즘**: 전체 그리드 스캔 → 모든 매치 수집 → 우선순위 정렬 → 소비 추적 `used[][]` 배열로 상위 패턴이 하위 패턴의 보석을 선점하는 방식. O(n²) 스캔이지만 8×8 = 64칸이므로 프레임당 부담 0.
- **캐스케이드 tween 체인**: 매칭→제거 애니메이션→낙하 tween→리필→재매칭 체크를 tween `onComplete` 콜백으로 체이닝. setTimeout 0건으로 프레임 독립적 동작 보장.
- **DDA(Dynamic Difficulty Adjustment)**: failStreak 기반으로 3연패 시 목표색 보석 출현 가중치를 조정하여 "거의 성공"하는 경험을 유도. localStorage에 failStreak/totalPlays를 기록하여 세션 간 연속성도 확보.
- **IX Engine 통합**: Input(키보드+마우스+터치 통합), Tween, Particles, Sound(Web Audio BGM+SFX 8종), Save(localStorage), AssetLoader(manifest.json 동적 로드)를 플랫폼 엔진에서 일괄 제공받아 보일러플레이트를 최소화.
- **초기 보드 안전 생성**: 3매치 없음 + 유효 이동 1개+ BFS 검증을 최대 100회 시도하고, 초과 시 안전 패턴(격자형 색 배치)으로 폴백. "불가능한 시작 보드" 원천 차단.

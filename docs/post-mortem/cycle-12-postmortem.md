# 미니 드리프트 레이서 — 포스트모템

## 한 줄 요약
InfiniTriX 최초의 레이싱 게임 — 탑다운 드리프트 물리, 5개 테마 트랙, 고스트 레코드까지 단일 HTML로 완성하고 1회차 리뷰에서 APPROVED를 달성했다.

## 무엇을 만들었나
탑다운 시점에서 관성 기반 차량을 조종하며 드리프트 포인트를 쌓고 랩 타임을 단축하는 아케이드 레이싱 게임이다. 핸드브레이크로 후미를 미끄러뜨리고 카운터 스티어링으로 각도를 유지하며 코너를 돌파하는 물리적 쾌감이 핵심이다. 드리프트 포인트가 부스트 게이지를 충전하고, 부스트로 직선을 가속하면 다음 코너 진입 속도가 올라가는 리스크-리워드 순환이 "한 바퀴만 더"를 부르는 중독 루프를 만든다.

초원, 사막, 설산, 도심 야경, 화산까지 5개 트랙이 각각 고유한 배경색·노면 효과·목표 시간을 갖고 있으며, 트랙별 별점(1~3성)과 고스트 레코드가 localStorage에 저장되어 자기 자신과의 기록 경쟁을 유도한다. 모래(마찰↑), 빙판(횡마찰↓), 물웅덩이(속도↓+조향반전), 용암(3초 정지)의 4종 노면 효과와 클린랩/드리프트킹/스피드데몬 3종 랩 보너스가 트랙마다 다른 공략법을 요구한다. 12사이클 만에 처음으로 도전한 레이싱 장르이지만, 기존 플랫폼 인프라(TweenManager, ObjectPool, SoundManager, TransitionGuard)가 무리 없이 적용되어 아키텍처 범용성을 다시 한번 확인한 사이클이었다.

## 잘 된 점 ✅
- **기획서 기능 100% 구현 + 1회차 APPROVED.** 코드 리뷰 전 항목 PASS, 브라우저 테스트 12/12 전항 PASS, 콘솔 에러 0건. 5개 트랙, 드리프트 물리, 콤보 시스템, 부스트 순환, 별점, 고스트 레코드, 4종 노면 효과, 3종 랩 보너스, 물웅덩이 조향 반전까지 빠짐없이 구현했다.
- **순수 Canvas 드로잉 달성.** 코드 내 `new Image()`, `fetch`, SVG 참조가 완전히 0건이다. 12사이클간 이어진 에셋 참조 문제에서 "코드 레벨 클린"을 최초로 완벽 달성했다.
- **드리프트 물리의 완성도가 높다.** 핸드브레이크 → 횡방향 마찰 감소 → driftAngle 누적 → 관성 슬라이드의 흐름이 자연스럽고, 부스트 순환(DP 500 → 게이지 충전 → MAX_SPEED 550 가속)도 기획 의도대로 동작한다.
- **고스트 시스템이 인상적이다.** 0.1초 간격 기록 → localStorage 저장 → 보간 재생 → 반투명 렌더링으로 서버 없이 비동기 기록 경쟁을 구현했다.
- **메모리 관리가 깔끔하다.** 연기 파티클 80개 ObjectPool, 타이어 자국 200개 링 버퍼, 레이스 리셋 시 전체 초기화 + smokePool.clear()까지 철저하다.
- **모바일 터치 대응이 완벽하다.** 7개 터치 버튼 모두 44px 이상, 멀티터치 지원(가속+조향 동시), 히트영역 확장(+12px/+20px), 스크롤/줌 방지까지 빈틈 없다.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **assets/ 디렉토리가 여전히 물리적으로 존재한다.** 코드에서는 참조하지 않지만 manifest.json + SVG 9개 파일이 배포에 포함된다. 12사이클째 반복되는 이 문제는 CI/pre-commit 훅 없이는 근본 해결이 불가능하다는 결론이 더욱 명확해졌다.
- **트랙 선택 화면 UX.** 480px 이하 뷰포트에서 5개 카드(총 780px)가 넘치는 문제가 있다. 좌우 스와이프 또는 카드 축소를 검토할 만하다. (터치 탭 선택은 정상 동작)
- **시간이 있었다면:** 리플레이 저장/공유, AI 상대 차량, 트랙 에디터, 날씨 효과(비/눈 시 노면 마찰 변화), 탈것 커스터마이징 등을 추가했을 것이다.
- **다음에 같은 장르를 만든다면:** 트랙 데이터를 JS 배열 하드코딩 대신 JSON 분리하여 확장성을 확보하고, 서브스텝 물리(fixed timestep)로 고속 충돌 안정성을 높이겠다.

## 기술 하이라이트 🛠️
- **Catmull-Rom 스플라인 기반 트랙:** 웨이포인트 배열에서 Catmull-Rom 보간으로 매끄러운 곡선 트랙을 생성하고, `findNearestSegment()` ±30 범위 탐색으로 전체 순회 없이 효율적인 도로 이탈 판정을 수행한다.
- **lerp 카메라 + 전방 룩어헤드 + 셰이크 감쇠:** 고속 레이싱에서 시야 안정성과 역동적 연출을 동시에 달성. Cycle 11 플랫포머의 카메라 노하우가 장르를 넘어 재활용된 성공 사례다.
- **Web Audio 절차적 사운드 5종:** 엔진음(sawtooth), 스키드음(noise+bandpass), 부스트음(sawtooth sweep), 벽충돌(noise+lowpass), 랩클리어(3음 시퀀스)를 외부 파일 없이 코드로 합성한다.
- **니어미스 판정:** 도로 경계 10px 이내 접근 시 보너스 배율을 적용하여, 벽에 가깝게 드리프트할수록 보상이 높아지는 리스크-리워드 피드백을 물리 레벨에서 구현했다.

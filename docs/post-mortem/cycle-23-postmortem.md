# 팬텀 시프트 — 포스트모템

## 한 줄 요약
빛과 그림자 이중 차원을 실시간으로 전환하며 프로시저럴 던전을 탐험하는 퍼즐-액션 로그라이크를 구현하여, 플랫폼의 puzzle+action 장르 공백을 해소했다.

## 무엇을 만들었나
팬텀 시프트는 "차원의 수호자"가 되어 빛의 세계와 그림자 세계가 겹쳐진 던전을 탐험하는 게임이다. Q키로 차원을 전환하면 벽이 사라지고 새로운 통로가 열리며, 적의 차원 속성(빛/그림자/혼합)에 맞춰 전환 타이밍을 판단하는 것이 핵심 긴장감이다. 차원 에너지를 관리하면서 "지금 전환할까, 보스전까지 아낄까?"를 매 순간 고민해야 한다.

15층 프로시저럴 던전 + 3보스(그림자 파수꾼/차원 직조자/그림자 군주) + 히든 스테이지(16층 빛의 기원)의 풍부한 콘텐츠 위에, SeededRNG 기반 랜덤 아이템/유물 조합과 영구 업그레이드 트리가 리플레이 가치를 더한다. 난이도 3종(쉬움/보통/어려움)과 한/영 이중 언어를 지원한다.

공간 퍼즐(두 차원의 지형을 머릿속에 겹쳐 최적 경로 탐색)과 실시간 전투 액션(회피 + 차원 전환 공격)이 결합된 하이브리드 장르로, 기존 퍼즐 게임(color-merge-puzzle 등)과 명확히 차별화된다.

## 잘 된 점 ✅
- **puzzle+action 장르 공백 최초 해소**: 플랫폼에 0개였던 이 조합을 성공적으로 채워, 23사이클 중 가장 독특한 장르 포지션을 확보했다.
- **5라운드 P0 버그 최종 해결**: Cycle 21 R1부터 5라운드 연속 미수정이었던 GAMEOVER→TITLE 전환 차단 버그가 `RESTART_ALLOWED` 화이트리스트 패턴으로 드디어 해결. 이 패턴은 앞으로 모든 게임의 표준이 될 것이다.
- **BFS 경로 검증(verifyPath)**: 프로시저럴 던전 생성에서 양 차원의 시작→출구 접근성을 BFS로 보장하여, "진행 불가능 맵 생성" 문제를 구조적으로 차단했다.
- **F1~F43 역대 최다 43건 피드백 매핑**: 23사이클간 축적된 모든 교훈을 기획서 §0에 2계층(검증 완료 요약 + 신규 상세)으로 분리 매핑. 피드백 매핑 분량 과다(F43) 문제도 해결.
- **코드 품질 11대 우수 사항**: 에셋 참조 0건, setTimeout 0건(12사이클 연속), 순수 함수 드로잉, TDZ 방지, ACTIVE_SYSTEMS 매트릭스, ObjectPool, REGION 1~10 구조 등 전 항목 PASS.
- **assets/ 미생성 6사이클 연속**: Cycle 18부터 시작된 에셋 제로 원칙이 완전히 정착.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **2차 리뷰 2회차까지 소요**: P0(GAMEOVER→TITLE)과 P2(스킬 버튼 48px 미달)가 초기 구현에서 미해결 상태로 제출되어 복수 라운드가 필요했다. RESTART_ALLOWED 패턴을 기획서 단계에서 명시했더라면 1차 통과가 가능했을 것이다.
- **GAMEOVER→TITLE P0 버그 5라운드 지속**: Cycle 21부터 반복된 이 버그는 STATE_PRIORITY 가드의 "역방향 전환" 예외 처리가 체계화되지 않았음을 보여준다. 앞으로 GAMEOVER/VICTORY 같은 종결 상태에서의 탈출 전환은 기획서에 명시적 화이트리스트로 반드시 포함해야 한다.
- **단일 파일 규모 지속 증가**: 2,400줄+ 단일 HTML 파일. REGION 주석 분리로 가독성은 확보했으나, 공용 엔진 모듈 분리가 23사이클째 지연 중이다. 한계를 완전히 넘어선 상태.
- **밸런스 자동 검증 미비**: 15층 × 3난이도 × 다수 유물/아이템 조합 공간에 대한 자동 밸런스 검증 수단이 없다. "특정 유물 조합이 압도적"이거나 "후반 층이 불가능"한지 코드 리뷰만으로는 판별 불가.
- **다중 이해관계자 피드백 프로세스**: 플래너·디자이너 피드백이 리뷰 라운드를 연장시키는 패턴이 3사이클 연속(Cycle 21~23) 재발. 통합 프로세스 확립이 시급하다.

## 기술 하이라이트 🛠️
- **이중 차원 맵 시스템**: 동일 그리드에 빛 레이어와 그림자 레이어를 겹쳐 관리. `isTileBlocked()` 4점 검사로 현재 차원의 지형만 충돌 판정에 적용하고, Q키 전환 시 차원 에너지 소모 + 쿨다운 검사를 `tryDimensionShift()`에서 처리.
- **SeededRNG 프로시저럴 던전**: 시드 기반 난수 생성으로 동일 시드에서 같은 던전을 재현. verifyPath() BFS 검증으로 양 차원 모두 시작→출구 접근성을 보장하는 안전한 절차적 생성.
- **RESTART_ALLOWED 패턴**: `beginTransition()`에서 STATE_PRIORITY 역방향 가드의 예외를 명시적 화이트리스트(`['GAMEOVER', 'VICTORY']`)로 관리. 5라운드간 미해결된 P0 버그의 근본 해결책이자, 향후 모든 게임에 적용할 표준 패턴.
- **ACTIVE_SYSTEMS 매트릭스**: 15개 상태별 활성 시스템을 명시적으로 정의하여, 특정 상태에서 tween/물리가 멈추는 류의 버그를 구조적으로 방지.
- **빛/그림자 분할 비주얼**: 타이틀 화면 좌우 분할 + 중앙 차원 균열 라인 + 빛(#FFD700)/그림자(#8B5CF6) 파티클 + 3px 스캔라인 이펙트로 이중 차원 콘셉트를 시각적으로 강렬하게 전달.

3. **프로시저럴 콘텐츠 밸런스 시뮬레이터**: 15층 × 3난이도 × 유물 조합을 자동 플레이하여 층별 클리어율/유물 선호도를 통계적으로 검증하는 도구 개발.

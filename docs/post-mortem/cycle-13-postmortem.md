# 미니 아이들 팜 — 포스트모템

## 한 줄 요약
빈 땅에서 작물·가축·가공으로 농장 제국을 키우는 아이들 시뮬레이션을 만들었고, 기획서 수치 100% 일치 + 에셋 제로를 달성했지만 터치 타겟 크기 미달로 NEEDS_MINOR_FIX를 받았다.

## 무엇을 만들었나
미니 아이들 팜은 6칸 그리드 농장에서 밀부터 시작해 가축, 가공품까지 10종 자원을 생산·판매하며 농장 제국을 키우는 아이들 시뮬레이션이다. 탭하면 수확 파티클이 터지며 즉시 골드를 얻고, 자동화 업그레이드를 구매하면 손을 놓아도 수익이 쌓인다. 프레스티지로 농장을 리셋하면 영구 성장 배율을 얻어 매 회차가 기하급수적으로 빨라지는 것이 장기 목표다.

이번 게임은 Cycle 11의 아이들 생산 파이프라인 노하우와 Cycle 8의 타이쿤 업그레이드 경제를 결합한 시도였다. 이전 Cycle 13에서 한글 워드 퀘스트가 index.html 미생성으로 실패한 뒤, 기획을 전면 변경하여 플랫폼 최초의 농장 아이들 장르에 도전했다. 3단계 농장 확장(밭→목장→가공), 6종 프레스티지 영구 업그레이드, 오프라인 수입, 마일스톤 5단계 등 기획서의 전체 범위가 충실히 구현되었다.

## 잘 된 점 ✅
- **기획서 수치 100% 일치**: 밀 3초/2G부터 스웨터 30초/50G까지 10종 자원, 프레스티지 별 공식 `floor(sqrt(totalEarned/1000))`, 오프라인 4시간 ×0.5 배율 등 CONFIG 객체의 30개+ 수치가 기획서와 완벽히 일치
- **에셋 제로 원칙 달성**: 13사이클 연속 재발했던 assets/ 디렉토리 문제를 드디어 극복. 100% Canvas + Web Audio, 외부 파일 0건, fetch/Image 사용 0건
- **코드 리뷰 16/16 전항 PASS**: 기능 완성도, 게임 루프, 메모리 관리, 모바일 터치, 상태 전환, 보안, 성능 등 모든 정적 분석 항목을 통과
- **브라우저 테스트 완벽 통과**: 콘솔 에러 0건, 타이틀→농장→수확→업그레이드→프레스티지→일시정지 전체 플로우 정상 동작
- **TransitionGuard + 순수 함수**: 5상태 × 6시스템 매트릭스가 코드 주석으로 명시되고 구현과 일치. `calcGrowTime()`, `calcPrestigeStars()` 등 핵심 함수가 모두 파라미터 기반 순수 함수

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **터치 타겟 48×48px 미달**: `CONFIG.MIN_TOUCH = 48`을 선언해놓고 실제 버튼 크기에 적용하지 않은 것이 가장 아쉬운 점. 일시정지(48×40), 사운드(40×40), 업그레이드 구매(80×36), 배치 메뉴(높이 44px) 등 다수 UI 요소가 WCAG AAA 기준 미달
- **SoundManager의 setTimeout 잔존**: 기획서 §5에서 "setTimeout 0건 목표"를 세웠지만, 사운드 시퀀싱용 setTimeout이 3건 남아 있다. Web Audio의 `oscillator.start(ctx.currentTime + delay)` 패턴으로 대체 가능
- **Cycle 13 전반의 혼란**: 한글 워드 퀘스트 실패 → 미니 아이들 팜 재기획이라는 사이클 내 피벗이 발생하여 game-id 추적이 복잡해졌다
- **시간이 있었다면**: 비료 효과의 시각적 연출 강화, 가축별 고유 애니메이션 다양화, 프레스티지 영구 업그레이드의 시각적 차별화를 추가했을 것

## 기술 하이라이트 🛠️
- **ObjectPool 이중 운용**: 파티클 80개 + 팝업 30개를 풀링하여 수확 시 대량 이펙트에서도 GC 스파이크 없이 안정적 동작
- **offscreen Canvas 배경 캐시**: 하늘 그라디언트 + 구름 + 태양 + 울타리를 1회 렌더 후 재사용. 매 프레임 복잡한 배경을 다시 그리지 않아 성능 확보
- **오프라인 수입 계산**: 접속 시 `Date.now()` 경과 시간 × 생산율 × 0.5 배율로 최대 4시간까지 오프라인 수입을 정산하고 환영 팝업 표시
- **롱프레스 연속구매 + 스와이프 탭 전환**: 500ms 후 100ms 간격 자동 반복 구매, 좌우 50px 스와이프로 탭 전환 — Cycle 11에서 검증된 아이들 모바일 UX 패턴을 재활용
- **세이브 호환성 보정**: 이전 버전 세이브 로드 시 누락 필드를 자동 보정하는 방어적 코딩으로 데이터 유실 방지

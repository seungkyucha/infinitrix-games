# 잉크 메이든 — 포스트모템

## 한 줄 요약
잉크 선화+수채화 아트의 메트로배니아를 단일 HTML로 구현하고, 14항목 셀프 QA 체크리스트 덕분에 역대 최초로 1차 리뷰 APPROVED를 달성했다.

## 무엇을 만들었나
**잉크 메이든**은 잉크 세계에 갇힌 소녀 '마리'가 대시→이중점프→벽타기를 순차 획득하며 봉인된 영역을 해금하는 횡스크롤 메트로배니아다. 잉크 숲·수묵 동굴·먹물 성·어둠의 방 4개 영역을 타일 기반 맵(약 24개 화면, 16×10 타일)으로 구성하고, 영역마다 보스가 HP 50% 기준으로 2페이즈 전환된다. "저 길은 아직 못 간다 → 능력 획득 → 되돌아오면 새 길이 열린다"는 메트로배니아 고유의 도파민 루프가 핵심이다.

5개 비밀방, 3단계 난이도(HP·적 배율·세이브포인트 간격 차등), S~C 등급 시스템과 시간보너스·노데미지보너스가 반복 플레이를 유도한다. Hollow Knight / Cuphead에서 영감받은 핸드드로우 수채화 에셋이 종이 텍스처 배경 위에서 7색 팔레트로 일관되게 표현된다. 패럴렉스 3레이어 배경과 어둠의 방 라디얼 라이팅이 영역별 분위기를 효과적으로 차별화한다.

## 잘 된 점 ✅
- **셀프 QA 프로토콜 효과 입증**: C1~C3 3사이클 연속 3차 리뷰라는 악순환을 마침내 끊었다. 기획서에 14항목 체크리스트 + 풀 플로우(로드→탐험→능력→보스→클리어→재시작) 셀프 QA를 의무화한 결과, 1차 리뷰부터 HIGH 항목 0건 APPROVED. 2차 리뷰는 교차 확인용이었고 코드 수정 없이 통과했다.
- **메트로배니아 핵심 구조 완전 구현**: 능력 기반 맵 차단(대시→벽 파괴, 이중점프→높은 절벽, 벽타기→수직 통로), 비밀방 탐색 보상, 세이브 포인트, 4보스 2페이즈까지 장르 필수 요소를 모두 담았다.
- **핸드드로우 에셋 일관성**: 캐릭터·적·배경·UI 전체에 잉크 선화+수채화 스타일을 통일. 리뷰에서 G-1 art-style 일관성, G-2 캐릭터 변형 동일 인물 모두 PASS.
- **전 검증 항목 통과**: IX 엔진 준수(A), 버튼 3방식(B), 재시작 3회(C), 플레이 완성도(D), 전환/Stuck 방어(E), 입력(F), 에셋 일관성(G) — 7개 카테고리 전체 PASS.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **에셋 포맷 혼재 (LOW)**: player 변형 에셋이 SVG와 PNG가 섞여 있다. 게임 동작에 영향은 없지만, 자동 검증 파이프라인으로 구조적 해결이 바람직하다.
- **모바일 세로 모드 빈 공간**: 16:10 가로 게임 특성상 세로 모드에서 여백이 발생한다. 가로 모드 유도 안내 UI가 있었다면 더 나았을 것이다.
- **최종 보스 난이도 급등**: 어둠의 붓 2페이즈에서 그림자 마리가 플레이어 기술을 모방하는 패턴은 인상적이나, 난이도 곡선이 가파르다. 중간 체크포인트나 패턴 힌트 연출이 추가됐으면 좋았겠다.
- **능력 조합 탐험 부족**: 대시+이중점프+벽타기를 조합해야만 접근 가능한 "메타 비밀방"이 있었다면 탐험의 깊이가 한층 늘었을 것이다.

## 기술 하이라이트 🛠️
**JSON 선언적 맵 시스템**: 24개 방을 JSON 데이터로 정의하고, exits 필드로 방 간 연결을 관리한다. 화면 경계 도달 시 자동 전환되며, 부서지는 벽(breakwall)의 편집 상태를 `_origMap`으로 백업·복원하여 재시작 시 완전한 맵 초기화를 보장한다. 절차적 생성(C1 BSP)과 달리 수작업 레벨 디자인의 품질을 유지하면서 확장성도 확보한 접근이다.

**능력 기반 진행 차단**: `abilities` 객체의 boolean 플래그로 입력 허용을 제어하고, 맵 타일 타입(4=breakwall)과 능력 보유 여부를 조합해 통과 가능 여부를 판정한다. 물리 파라미터 조절(대시: 무적+고정속도, 벽타기: 중력×0.3)만으로 각 능력을 구현해 코드 복잡도를 최소화했다.

**어둠의 방 라디얼 라이팅**: 최종 영역에서 전체 화면을 어둡게 덮고 플레이어 좌표 기준 원형 라디얼 그라데이션 마스크를 씌워 "플레이어 주변만 보이는" 연출을 구현했다. 단순한 기법이지만 긴장감 있는 분위기를 효과적으로 만들어냈다.

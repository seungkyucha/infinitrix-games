# 유령 탐정 (Spectral Sleuth) — 포스트모템

## 한 줄 요약
살해당한 탐정이 유령이 되어 투시·감응·시간역행 능력으로 단서를 모으고, 논리 퍼즐로 범인을 추리하는 1920년대 아르데코 미스터리 퍼즐 어드벤처를 완성했다.

## 무엇을 만들었나
"유령 탐정"은 플랫폼 최초 탐정 테마 게임이다. 플레이어는 억울하게 살해당한 명탐정 루시안 그레이가 되어, 투시(Clairvoyance)·감응(Empathy)·시간역행(Rewind) 세 가지 유령 능력으로 산 자들이 볼 수 없는 단서를 발견한다. 수집한 단서를 조합하여 "추리 체인"을 완성하고, 용의자의 증언에서 모순을 찾아 증거를 제시하는 Phoenix Wright식 대질 보스전으로 사건을 해결한다.

핵심 차별점은 **비전투 보스전**이다. 전투가 아닌 추리와 논증으로 보스를 이기는 구조는 플랫폼 내 혁신적 경험이며, puzzle+casual 장르 조합을 기존 2개에서 3개로 강화했다. 1920년대 아르데코 항구 도시 "미스트헤이븐"을 배경으로 한 비주얼은 가스등 앰버 조명과 시안 유령 시야의 대비, 지역별 날씨 효과(안개/비/눈/먼지/에테르 입자)로 분위기를 극대화한다. 이로써 8사이클 연속 서로 다른 장르 조합(#25~#32)을 달성했다.

## 잘 된 점 ✅
- **INIT_EMPTY 패턴 완전 정착**: Cycle 31에서 3차 리뷰까지 끌고 갔던 TDZ 크래시를 G 객체 빈 구조 초기화 패턴으로 원천 차단. 이번 사이클에서 TDZ 관련 이슈 0건.
- **에셋 규칙 완벽 준수**: ASSET_MAP/SPRITES/preloadAssets 코드 0건, orphaned SVG 전량 삭제(attempt 3에서 8개 정리 완료), thumbnail.svg + manifest.json만 잔존하는 최소 상태 달성.
- **모바일 입력 완전 구현**: 가상 조이스틱 + 능력 버튼 + 더블탭(상호작용) + 롱프레스(능력 사용) + 증거 카드 탭 + 검증 버튼까지, 키보드 없이 전 흐름 조작 가능. 13개 모바일 항목 전부 PASS.
- **프로시저럴 사운드 역대 최다**: Web Audio API 기반 SFX 8종 + BGM 4종을 외부 오디오 파일 0건으로 달성.
- **18개 상태 완전 매핑**: ESCAPE_ALLOWED + ACTIVE_SYSTEMS 매트릭스가 18개 상태 전체를 무결하게 관리. beginTransition 단일 정의로 데드코드 0건.
- **코드 품질**: SeededRNG 100%(Math.random 0건), setTimeout 0건, alert/confirm/prompt 0건, eval 0건, hitTest() 통합, 순수 함수 패턴, 다국어(ko/en) 완전 지원.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **3차 리뷰까지 소요**: attempt 1~2에서 orphaned SVG 8개 잔존(P1-R), beginTransition 이중 정의(P2), RESTART_ALLOWED 데드코드(P3)가 지적되어 3번째 제출에서야 APPROVED. 기획 단계에서의 에셋 파일 관리와 코드 정리를 프리 리뷰 게이트에 더 강력히 통합해야 한다.
- **밸런스 미검증**: 5구역 × 3사건 × 3능력 × 8도구 × DDA의 조합 공간을 코드 리뷰만으로 검증할 수 없다. 퍼즐 난이도를 단서수/시간제한/오답허용 3변수로 축소한 것은 좋은 접근이었으나, 자동 시뮬레이션 없이는 "특정 구역이 불가능"하거나 "능력 하나만 올리면 끝"인 문제를 사전에 잡을 수 없다.
- **공용 엔진 미분리 32사이클째**: TweenManager, ObjectPool, SoundManager, InputManager 등이 여전히 게임마다 복붙되고 있다. 10 REGION 구조로 의존 방향은 정리되었으나 실제 모듈 추출은 착수하지 못했다.
- **소형 디스플레이(320px) 심층 검증 미완**: 5단계 뷰포트 매트릭스를 기획에 포함했으나, 320px 극소형에서의 증거 패널/추리 보드 UI 겹침 여부는 실기기 검증이 부족하다.

## 기술 하이라이트 🛠️
- **아르데코 비주얼 순수 Canvas 구현**: 기하학적 건축 실루엣, 유령 글로우 오라(5프레임 잔상 트레일), 가스등 앰버 조명, 별+달 날씨 효과, 전환 애니메이션 4종(fade/wipe/slide/dramatic)을 에셋 0건 코드 드로잉으로 달성. 11색 팔레트(딥 네이비 #0B1426, 에테리얼 시안 #00D4FF, 앰버 골드 #FFB800 등)가 1920년대 유령 세계를 효과적으로 표현.
- **유령 능력 시각 모드 3종**: 투시(밝기↑ 채도↓ 시안 틴트), 감응(대상 주변 감정 색상 오라), 시간역행(세피아 톤 + VHS 노이즈 + 역재생 파티클). Canvas 필터와 컴포지팅으로 능력마다 완전히 다른 시각 경험 제공.
- **transProxy 기반 tween 전환 시스템**: beginTransition 단일 정의 + 4종 전환 효과를 프록시 패턴으로 깔끔하게 관리. 이전 사이클의 이중 정의 문제를 구조적으로 해소.
- **devicePixelRatio 대응**: `canvas.width = w * dpr` + `ctx.setTransform(dpr, ...)` 패턴으로 고해상도 디스플레이에서도 선명한 렌더링 보장.

# 글리프 래버린스 (Glyph Labyrinth) — 포스트모템

## 한 줄 요약
고대 유적 미궁을 탐험하며 글리프를 수집·조합하여 새 능력을 해금하고, 환경 퍼즐과 보스전을 공략하는 퍼즐 메트로이드바니아를 순수 Canvas로 구현했다.

## 무엇을 만들었나
글리프 래버린스는 5개 바이옴(화염 신전, 빙결 동굴, 고대 숲, 심연 폐허, 천공 탑)과 히든 바이옴 "시간의 균열"로 구성된 총 26개 방의 탑다운 메트로이드바니아다. 플레이어는 고대 문명 "아르카나"가 남긴 글리프를 수집하고, 5종 기본 글리프 + 15종 조합 글리프로 총 20개 능력을 해금하며, 이전에 접근할 수 없던 비밀 경로를 열어나가는 "아하!" 모먼트가 핵심이다.

보스전은 단순 액션이 아닌 환경 퍼즐과 전투의 결합이다. 바이옴 환경을 활용하고 글리프 능력을 창의적으로 사용해야 약점이 노출되는 패턴 관찰→환경 활용→타격의 사이클을 거친다. 3단 난이도(탐험가/모험가/전설), 콤보 시스템, SeededRNG 기반 프로시저럴 방 배치로 매 플레이가 다른 경험을 제공하며, arcade+puzzle 장르 조합의 빈도를 1→2개로 강화했다.

## 잘 된 점 ✅
- **3차 리뷰에서 100% 해결률 달성**: 2차 리뷰의 P0 2건, P1 2건, P2 1건을 3차에서 전부 해결하여 APPROVED. assets/ 불법 SVG 8개 삭제, 모바일 글리프 슬롯 터치 버튼 5개 추가, 일시정지 터치 버튼 구현까지 꼼꼼히 마무리했다.
- **모바일 완전 대응**: 가상 조이스틱 + 9개 터치 버튼(공격/상호작용/일시정지/대시 + 글리프 슬롯 5개)으로 키보드 없이도 완전한 플레이 가능. 모든 터치 타겟이 `Math.max(CONFIG.TOUCH_MIN_TARGET, ...)` 패턴으로 최소 48px 보장(F11 준수).
- **외부 의존성 완전 제로**: 외부 CDN 0건, Google Fonts 0건, new Image() 0건, alert/confirm/prompt 0건. 시스템 폰트 스택만 사용. assets/ 디렉토리 정리 완료(manifest.json + thumbnail.svg만 존재)로 F1 9사이클 연속 준수.
- **기획서 피드백 매핑 54건(F1~F54) 충실 반영**: Cycle 24 포스트모템 기반 신규 피드백 6건(F49~F54)을 포함하여 역대 최다 피드백을 선제적으로 반영. 특히 에셋 코드 원천 차단(F50), 외부 링크 자동 검증(F49) 등이 초회부터 P0 이슈를 예방했다.
- **전체 게임 플로우 안정성**: TITLE→DIFF_SELECT→EXPLORE→COMBAT→GAMEOVER→TITLE 전체 사이클이 회귀 없이 정상 동작. 브라우저 테스트 전 항목 PASS, 콘솔 에러/경고 0건.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **3라운드 리뷰 소요**: 1차에서 즉시 APPROVED를 받지 못하고 3차까지 걸렸다. assets/ 불법 SVG 잔존과 모바일 글리프 슬롯 미구현이 P0으로 잡혔다. 스모크 테스트 게이트(F15, F52)를 더 엄격하게 적용했다면 1~2차에서 해결 가능했을 것이다.
- **STATE_PRIORITY 데드코드 발생**: 기획서 F6에서 설계한 STATE_PRIORITY가 실제 코드에서 활용되지 않고 데드코드로 잔존했다. 기획 의도와 구현의 불일치는 여전히 주의가 필요하다. 설계 시점에서 "이 코드가 실제로 호출되는 경로"를 명시하는 습관이 필요하다.
- **밸런스 자동 검증 미비**: 5개 바이옴 × 5보스 × 20글리프 조합의 밸런스를 코드 리뷰만으로는 완전히 검증할 수 없다. F51에서 수치화를 시도했지만, 실제 플레이 데이터 없이는 "특정 글리프 조합이 지배적"이거나 "특정 보스가 불가능"한 상황을 사전에 잡기 어렵다.
- **공용 엔진 모듈 분리 25사이클째 지연**: TweenManager, ObjectPool, TransitionGuard, SoundManager, touchSafe(), ScrollManager 등의 copy-paste가 25개 게임에서 반복되고 있다. 유지보수 한계가 완전히 초과된 상태이며, 이번 사이클에서도 착수하지 못했다.

## 기술 하이라이트 🛠️
- **순수 Canvas 메트로이드바니아 렌더링**: 5개 바이옴의 고유 색상 팔레트(화염 오렌지레드, 빙결 딥스카이블루, 고대숲 라임그린, 심연 보라, 천공 금빛)를 Canvas 코드 드로잉만으로 표현. 글리프 파티클, 타이틀 다이아몬드 격자, 그라디언트 원 등 비주얼 이펙트가 에셋 없이 구현됐다.
- **SeededRNG 프로시저럴 방 배치 + BFS 도달 가능성 검증(F54)**: 매 플레이마다 다른 미궁 배치를 생성하면서, BFS 알고리즘으로 모든 방이 도달 가능한지 자동 검증하여 "불가능한 맵" 문제를 원천 차단했다.
- **DDA(Dynamic Difficulty Adjustment) + 3단 난이도 + 콤보 시스템**: 플레이어 실력에 따라 난이도가 동적으로 조정되고, 콤보 배수(COMBO_MULT 배열 + COMBO_WINDOW)가 적 처치 연쇄에 보상을 제공하는 다층 난이도 구조.
- **모바일 터치 시스템 확장**: 기존 4개 액션 버튼에 글리프 슬롯 5개를 추가하여 총 9개 터치 버튼 체계 완성. 일시정지 화면에도 전용 터치 버튼("이어하기"/"타이틀로")을 추가하여 모바일에서의 UI 데드엔드를 완전히 해소했다.

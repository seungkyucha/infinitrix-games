# 잿빛 요새 (Ashen Stronghold) — 포스트모템

## 한 줄 요약
포스트아포칼립스 좀비 서바이벌 타워디펜스 로그라이트를 2회 리뷰 만에 APPROVED로 완성했다.

## 무엇을 만들었나
낮에는 폐허를 탐색하며 자원과 생존자를 수집하고, 밤에는 바리케이드와 포탑을 배치해 좀비 무리를 막아내는 서바이벌 타워디펜스 로그라이트다. Plants vs Zombies의 배치 전략, They Are Billions의 서바이벌 긴장감, Slay the Spire의 로그라이트 선택지를 한 게임에 녹여냈다.

핵심 루프는 **주간 탐색(60초) → 방어 준비(30초) → 야간 웨이브(~90초)**로 구성된다. 3구역 × 3야간 = 9 메인 스테이지에 보스 3종(스포어 타이탄, 아이언 리퍼, 네크로 퀸), 업그레이드 트리 3축(방어/공격/탐색) × 5단계, 유물 13종(일반/희귀/에픽), DDA 4단계 난이도 보정까지 — 플랫폼 최초 좀비 포스트아포칼립스 테마로 action+strategy 장르의 11사이클 미사용 공백을 해소했다.

## 잘 된 점 ✅
- **1회차 지적사항 3건 모두 2회차에서 완벽 수정**: 자원 표시 부동소수점(`Math.floor` 적용), R 버튼 터치 타겟(r=24, 직경 48px), onBossDefeated 전환 가드 충돌(직접 상태 전환) — 수정 회귀 0건.
- **에셋 21개 전부 정상 로드 + Canvas 폴백 100% 완비**: Gemini API PNG 에셋이 manifest.json 기반으로 안정적으로 로드되면서도, 모든 렌더 함수에 Canvas 폴백이 존재하여 에셋 실패에도 게임이 동작한다.
- **코드 품질 전항 PASS**: keydown preventDefault, rAF + delta time, 터치 이벤트, TRANSITION_TABLE 7상태, localStorage 최고점, canvas resize + DPR, 외부 CDN 0건, Math.random 0건(SeededRNG 완전 사용), alert/confirm 0건.
- **비주얼 분위기 완성도**: 폐시가지 배경 에셋, 야간 비네트, 비 효과, 스캔라인, 좀비/보스 스프라이트가 포스트아포칼립스 세계관을 효과적으로 전달한다.
- **Puppeteer 전체 플로우 완주**: TITLE → MAP → DAY_EXPLORE → NIGHT_PREP → NIGHT_WAVE → GAMEOVER → TITLE 전 경로를 자동 테스트로 검증. 콘솔 에러 0건.

## 아쉬웠던 점 / 개선 가능성 ⚠️
- **monkey-patch 확장 구조 유지(INFO-1)**: 확장 시스템들이 IIFE 래핑으로 기존 함수를 감싸는 구조가 그대로 남아있다. 기능상 문제는 없으나, 코드 가독성과 유지보수성 측면에서 향후 리팩토링이 권장된다.
- **beginTransition 가드의 과도한 차단(INFO-2)**: core.hp ≤ 0일 때 GAMEOVER 외 전환을 차단하는 가드가 DAY_EXPLORE/NIGHT_PREP 중 인위적 hp=0 설정 시 전환을 막는다. 정상 플레이에서는 발생하지 않지만, 디버깅 시 혼란을 줄 수 있다.
- **Phase 2 미구현**: 기획서의 구역 4~5(군부대, 지하벙커), 히든 스테이지 2개, 업그레이드 Lv4~5, 날씨/시간대 연출은 Phase 1 MVP에 포함되지 않았다. 시간이 있었다면 콘텐츠 볼륨을 확장할 수 있었을 것이다.
- **밸런스 자동 검증 부재**: 3구역 × 3야간 × 보스 3종 × 유물 13종 × DDA 4단계의 조합 공간이 넓지만, 코드 리뷰만으로 밸런스를 판별할 수 없다. DPS 캡 200%/시너지 캡 150%를 설정했으나 극단 빌드의 실제 클리어 가능 여부는 미검증.

## 기술 하이라이트 🛠️
- **BFS 좀비 경로 탐색 + 배치 차단 방지**: 좀비 AI가 코어까지 BFS로 최단 경로를 계산하고, 플레이어가 방어물을 배치할 때 `bfsPathExists()` 검증으로 경로를 완전히 막는 배치를 거부한다. 직선 폴백도 존재하여 경로 계산 실패 시에도 게임이 중단되지 않는다.
- **SeededRNG 완전 사용**: `Math.random()` grep 결과 0건. 프로시저럴 야간 웨이브, 아이템 배치, 유물 드롭 등 모든 랜덤 요소가 시드 기반으로 동작하여 리플레이 일관성을 보장한다.
- **`_ready` 플래그 TDZ 방어 패턴**: Engine 생성자 내 콜백(onResize 등)이 미완료 engine 변수를 참조하는 Cycle 39의 P0 크래시를 `_ready` 플래그 + 모든 콜백 진입부 가드로 완벽 차단. 이 패턴이 사이클 39~40 교훈의 핵심이었고, 이번 사이클에서 처음으로 실전 검증되었다.
- **DDA 4단계 동적 난이도**: 플레이어 실력에 따라 좀비 HP/속도/수량/자원 보상이 4단계로 조절되어, 초보자도 후반까지 도달할 수 있고 숙련자에게도 도전 요소가 유지된다.
- **다국어(ko/en) + L키 토글**: 타이틀 화면에서 L키로 한국어↔영어 전환. 모든 UI 텍스트가 L 객체에 ko/en 쌍으로 관리된다.

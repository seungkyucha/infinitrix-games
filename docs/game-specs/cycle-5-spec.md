## 1. 게임 개요 및 핵심 재미 요소

**물들인 하늘**은 수채화로 물들인 하늘 세계가 어둠의 먹구름에 잠식당하는 이야기 속에서, 빛의 수호자 '아리아'가 빽빽한 탄막을 회피하며 4명의 타락한 화가를 정화하는 **종스크롤 탄막 슈터**이다.

Ori / Gris에서 영감받은 페인터리 수채화 배경 위로 극좌표 기반의 아름다운 탄막 패턴(나선형, 방사형, 파동형, 꽃잎형)이 펼쳐지며, **시각적 아름다움과 극한의 긴장감이 공존**하는 것이 핵심 차별점이다.

### 세계관
천상의 **채색 정원**은 네 명의 화가가 각자의 시간대를 그리며 하늘을 물들이는 낙원이었다. 어느 날 정원의 심장에 먹구름이 스며들어 화가들을 타락시켰다. 빛의 정령 '아리아'는 새벽→석양→별밤을 거쳐 먹구름 성채에 도달해 어둠의 근원을 정화해야 한다.

### 핵심 재미 3요소

1. **탄막 회피의 쾌감** — 화면을 메운 탄막 사이 좁은 틈을 꿰뚫는 아슬아슬한 회피. 히트박스는 캐릭터 중앙 4×4px로 극소화하여 "불가능해 보이는 패턴도 통과 가능"
2. **패턴 암기와 공략** — 보스마다 고유한 극좌표 탄막 패턴. 처음엔 불가능해 보이지만 패턴을 읽으면 안전 경로가 보이는 "유레카 모먼트"
3. **파워업 빌드의 전략** — 스테이지 사이 3택 파워업으로 빌드 구성. 공격형/방어형/특수형 조합 시 시너지 발동

### 30초 후크

> 게임 시작 → 수채화 새벽 하늘 + 빛 파티클 → 첫 적 웨이브 등장 → 화려한 꽃잎 탄막이 눈앞에 → 좁은 틈을 빠져나가며 그레이즈 보너스 → "한 판 더!"

### 레퍼런스
- **Touhou Project** — 종스크롤 탄막 패턴 설계의 교과서
- **Enter the Gungeon** — 탄막 + 로그라이트 빌드 다양성
- **Ori and the Blind Forest** — 페인터리 2D 발광 캐릭터 + 대기감 조명

## 2. 게임 규칙 및 목표

### 게임 구조
```
타이틀 → 난이도 선택(수채/먹물/진묵) → [스테이지 1~4: 웨이브 → 보스] → 클리어/게임오버 → 결과 → 메타 업그레이드 → 타이틀
```

### 기본 규칙
- 플레이어 HP: 이지 5 / 노멀 3 / 하드 1 (원히트킬)
- 피격 시 **2초 무적** + 화면 플래시
- 폭탄(Bomb): 화면 내 적탄 전소거 + 2초 무적. 이지 5개 / 노멀 3개 / 하드 1개
- 보스 격파 시 HP 1 회복 + 폭탄 1 보충
- 4스테이지 모두 클리어 시 게임 클리어

### 승리 조건
- 4보스 전원 격파 (최종 보스 포함)

### 패배 조건
- HP 0 → 게임오버 → 결과 화면 → 획득 "빛의 조각"으로 메타 업그레이드

### 스테이지 구조

| 스테이지 | 테마 | 배경 색조 | 길이 | 보스 |
|----------|------|-----------|------|------|
| 1. 새벽 하늘 | 파스텔 새벽 | 연분홍+하늘색 | 60초 | 안개 화가 |
| 2. 석양 들판 | 따뜻한 석양 | 주황+보라 | 75초 | 노을 무희 |
| 3. 별빛 바다 | 깊은 밤바다 | 남색+청록 | 90초 | 심해 거인 |
| 4. 먹구름 성채 | 어둠의 폭풍 | 검정+짙은보라 | 90초 | 먹구름의 화가 |

> ⚠️ 단일 HTML 스코프: 4스테이지 × (웨이브 3~4회 + 보스전). 탄막 패턴은 JSON 선언적 데이터로 정의.

## 4.5. 아트 디렉션 (Art Direction)

### 스타일 정의
- **아트 스타일 키워드**: `painterly-2d`
- **전체 cue**: "Painterly digital 2D illustration: visible brushstrokes, rich atmospheric lighting (Ori / Gris feel)."
- 모든 에셋은 이 아트 디렉션을 엄격히 따름
- **포맷**: SVG 단일 포맷 (혼재 금지 — C4 학습)

### 핵심 원칙
1. **수채화 붓터치**: 모든 오브젝트에 디지털 붓터치 질감. 날카로운 벡터 라인 지양
2. **대기감 조명**: 스테이지별 색 온도+빛 방향. 역광/림라이트로 실루엣 강조
3. **색상 대비 가독성**: 배경(부드러운 톤) vs 탄막(선명한 대비) → 탄막이 항상 구분
4. **파티클과 빛**: 빛 파티클, 수채 번짐 트레일로 움직임에 활기

### 레퍼런스 게임
1. **Ori and the Blind Forest** — 발광 캐릭터 + 대기감 배경 레이어링
2. **Gris** — 수채화 색조 전환 + 감성적 비주얼

### 비주얼 규칙
- 탄막은 **꽃잎/잉크방울/빛 입자** 형태 — 위협적이면서도 아름다움
- 보스는 **거대한 정령/화가** — 위엄 있고 신비로운 존재
- UI는 **부드러운 붓터치 프레임** 모티프

## 5. 핵심 게임 루프 (60fps 기준)

### 매 프레임 업데이트 순서
```
1. Input 처리
   ├─ 이동 벡터 계산 (Shift 시 속도 ×0.4)
   ├─ 발사 처리 (자동연사 8발/초)
   ├─ 폭탄 입력 체크
   └─ 씬 전환 직후 200ms 입력 무시

2. 스크롤 업데이트
   ├─ 배경 3레이어 패럴렉스 (원경 ×0.3, 중경 ×0.6, 근경 ×1.0)
   └─ 스테이지 진행도에 따른 스크롤 속도

3. 스폰 로직
   ├─ 적 웨이브 타이밍 (JSON 데이터 기반)
   ├─ 적 오브젝트 풀 acquire
   ├─ 파워업 드롭 판정
   └─ 보스 등장 트리거

4. 엔티티 업데이트
   ├─ 플레이어: 이동 (경계 16px 마진) + 자동 발사
   ├─ 아군 탄: 전진 → 화면 밖 release
   ├─ 적: AI 이동 + 탄막 발사 (극좌표 패턴)
   ├─ 적 탄: 이동 → 화면 밖 release
   └─ 아이템: 하강 이동

5. 충돌 판정 (공간 해싱 64×64)
   ├─ 아군 탄 vs 적 → HP 감소, 사망 시 release+이펙트
   ├─ 적 탄 vs 플레이어 히트박스(4×4px) → 피격
   ├─ 적 탄 vs 그레이즈 범위(20px) → 그레이즈 점수
   ├─ 적 본체 vs 플레이어 → 피격
   └─ 아이템 vs 플레이어 → 수집

6. 화면 효과
   ├─ 피격: 화면 흔들림 (5px, 0.2초)
   ├─ 폭탄: 화면 플래시 (백색 0.3초)
   ├─ 보스 등장: 줌 연출
   └─ 그레이즈: 미세 슬로우 (0.05초, dt×0.5)

7. 렌더링
   ├─ 배경 3레이어 패럴렉스
   ├─ 적 탄막 (활성 탄만)
   ├─ 적 엔티티
   ├─ 아이템
   ├─ 아군 탄
   ├─ 플레이어 (저속 시 히트박스 빛 원)
   ├─ 이펙트/파티클
   └─ UI (HP, 폭탄, 점수, 그레이즈, 보스 HP바)
```

## 11. 씬 구성

| 씬 | 내용 | 전환 |
|----|------|------|
| TITLE | 타이틀 + 시작/성소/조작법 | → DIFFICULTY/SANCTUARY/CONTROLS |
| DIFFICULTY | 수채/먹물/진묵 선택 | → GAME |
| GAME | 종스크롤 전투 | → BOSS_INTRO/POWERUP/RESULT |
| BOSS_INTRO | 보스 등장 (1.5초) | → GAME |
| POWERUP | 3택 파워업 선택 | → GAME (다음 스테이지) |
| RESULT | 점수·등급·빛의 조각 | → SANCTUARY/TITLE |
| SANCTUARY | 메타 업그레이드 | → TITLE |
| CONTROLS | 조작법 안내 | → TITLE |
| PAUSE | 계속/재시작/타이틀 | → GAME/TITLE |

> ⚠️ 모든 전환 시 **200ms 입력 무시**
> ⚠️ 동적 버튼 씬(POWERUP, SANCTUARY)은 enter()에서 버튼 빌드 필수

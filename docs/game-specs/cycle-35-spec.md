# 어비스 다이버 (Abyss Diver) — 사이클 #35 게임 기획서

> **1페이지 요약**: 고대 해저 문명 "아비소스"의 비밀을 파헤치는 심해 탐험가가 **적대적 심해 환경**을 실시간으로 돌파하는 **아케이드 퍼즐 플랫포머**. Level Devil(Poki #1)의 "레벨 자체가 적"이라는 핵심 메카닉을 심해 테마로 재해석 — 바닥이 무너지고, 해류가 반전되고, 수압이 조작을 왜곡하고, 생물발광 유인등이 함정을 숨긴다. 5개 바이옴 × 3스테이지 = 15 기본 스테이지 + 바이옴 보스 5종 + 히든 심연 스테이지 3개 = **총 23 스테이지**. 잠수복 업그레이드 트리(3갈래 × 5단계) + 도구 6종 언락 영구 진행 + SeededRNG 프로시저럴 함정 배치로 높은 리플레이 가치. **arcade+puzzle 조합으로 10사이클 최장 미사용(#25 이후) 해소 + 2번째 장르 순환 시작.** 플랫폼 최초 심해 테마, 바이오루미네선스·수압 역학 Canvas 비주얼 신규 도입.

> **MVP 경계**: Phase 1(핵심 루프: 잠수→퍼즐 돌파→탈출/보스, 바이옴 1~3 + 보스 3체 + 기본 도구 3종 + 업그레이드 트리 Lv1~3) → Phase 2(바이옴 4~5 + 보스 2체 + 히든 심연 3스테이지 + 전체 내러티브 + 도구 6종 + 업그레이드 Lv4~5 + 타임어택 랭킹). **Phase 1만으로도 완전한 게임 경험 제공 필수.**

## §1. 게임 개요 및 핵심 재미 요소

### 핵심 컨셉
"심해 자체가 적이다" — Level Devil의 "적대적 레벨" 메카닉을 심해로 옮겨 **환경이 능동적으로 플레이어를 방해하는** 아케이드 퍼즐 플랫포머.

### 핵심 재미 요소 5가지
1. **배신의 환경**: 안전해 보이는 산호가 무너지고, 바닥이 열리고, 해류가 갑자기 반전. "이 바닥은 안전할까?" 매 순간 긴장감
2. **수압 역학**: 깊이에 따라 이동 속도 감소·점프 높이 변화·시야 축소. 물리 법칙 자체가 퍼즐 요소
3. **바이오루미네선스**: 생물발광이 유일한 빛 — 유인등이 함정을 숨기고, 발광 패턴이 안전 경로를 힌트로 제공
4. **도구 퍼즐**: 음파탐지(숨겨진 경로 탐지), 전기충격(적 마비+기계 활성화), 조명탄(어둠 속 시야 확보) 조합
5. **즉사 & 즉부활**: Level Devil처럼 즉사 후 즉시 재시작. "한 번 더" 중독성

### 레퍼런스 분석 (Level Devil → Abyss Diver 변환)

| Level Devil 요소 | Abyss Diver 적용 | 차별화 |
|-----------------|-----------------|--------|
| 바닥 무너짐 | 산호초 붕괴, 해저 지반 균열 | 수중 물리(천천히 침하) |
| 벽 닫힘 | 해저 동굴 수축, 거대 조개 닫힘 | 틈새 빠져나가기 퍼즐 |
| 조작 반전 | 수압 조작 왜곡 (좌우 반전, 부력 변화) | 깊이별 점진적 강화 |
| 점프 변형 | 부력 변화 (무중력→초중력) | 산소 잔량과 연계 |
| 200 레벨 | 23 스테이지 (질적 밀도 우선) | 보스전+도구 퍼즐+내러티브 |

## §2. 게임 규칙 및 목표

### 최종 목표
고대 해저 문명 아비소스의 핵심에 도달하여 문명 멸망의 비밀을 밝혀내기.

### 스테이지 목표
각 스테이지의 **탈출구(심해 게이트)**에 도달하면 클리어. 탈출구 위치는 고정이나 도달 경로의 함정 배치가 SeededRNG로 변동.

### 사망 조건
- 함정 접촉 (즉사): 가시 산호, 독 해파리, 열수구 분출
- 수압 게이지 소진: 깊이별 수압이 지속 데미지. 업그레이드로 내구도 증가
- 산소 소진: 스테이지별 제한 시간(산소). 산소 버블로 보충 가능
- 적대적 해양 생물: 앵글러피시, 전기뱀장어 등 패턴 기반 이동

### 스테이지 구조 (5 바이옴 × 3스테이지 + 보스 5 + 히든 3 = 23)

| 바이옴 | 스테이지 | 환경 특성 | 핵심 함정 | 보스 |
|--------|---------|----------|----------|------|
| 1. 산호 정원 | 1-1~1-3 | 밝고 다채로운 산호, 약한 해류 | 붕괴 산호, 거품 함정, 위장 아네모네 | 앵글러피시 군주 |
| 2. 심해 동굴 | 2-1~2-3 | 어둠+생물발광, 중간 수압 | 동굴 수축, 종유석 낙하, 유인등 함정 | 거대 문어 현자 |
| 3. 열수 분출구 | 3-1~3-3 | 고온·적색 조명, 강한 해류 | 열수 간헐천, 용암류, 기포 상승류 | 화산 가디언 |
| 4. 해구 심연 | 4-1~4-3 | 극어둠·초고수압, 중력 왜곡 | 수압 조작왜곡, 블랙스모커, 심해어 습격 | 심해 해파리 여왕 |
| 5. 아비소스 유적 | 5-1~5-3 | 고대 기술+생물발광, 퍼즐 집약 | 기계 함정, 룬 퍼즐 잠금장치, 중력 역전 | 아비소스 핵심 |
| 히든. 심연의 균열 | H-1~H-3 | 모든 환경 요소 복합 | 랜덤 복합 함정 | — |

### 히든 스테이지 해금 조건
- H-1: 바이옴 1~3 전스테이지 산소 50%+ 잔량 클리어
- H-2: 바이옴 4 전스테이지 사망 0회 클리어
- H-3: 모든 문명 단서(15개) 수집 완료

## §7. 핵심 시스템 상세

### §7.1 수중 물리 엔진

#### §7.1.1 기본 물리
```
수평 이동: vx += inputDir * SWIM_ACCEL * depthFactor
수평 감속: vx *= WATER_DRAG (0.92)
수직 (부력): vy += GRAVITY - BUOYANCY * buoyancyMod
상승: vy -= SWIM_UP_FORCE * holdDuration (최대 MAX_UP_FORCE)
급강하: vy += DIVE_FORCE
```

**깊이 팩터 (depthFactor)**: 깊이가 증가할수록 감소
```
depthFactor = max(0.4, 1.0 - depth * DEPTH_DRAG_COEFF)
```

#### §7.1.2 해류 시스템
각 스테이지에 0~3개의 해류 영역. 해류는 방향+세기로 정의:
```
currentForce = { dx: float, dy: float, strength: 0.0~1.0 }
// 다이버 위치가 해류 영역 내일 때:
vx += currentForce.dx * currentForce.strength * dt
vy += currentForce.dy * currentForce.strength * dt
```
- 해류 방향은 시간에 따라 서서히 변화 (sinewave 기반)
- 바이옴 3(열수)은 강한 상승 해류, 바이옴 4(심연)는 난류(랜덤 방향 전환)

#### §7.1.3 충돌 검출
AABB 기반 타일맵 충돌 + 원형 충돌(생물체):
```
tileCollision(entity, tilemap) → { top, bottom, left, right, resolved }
circleCollision(entityA, entityB) → { hit, normal, depth }
```
모든 충돌 함수는 순수 함수 — 전역 참조 0건 (F9).

Phase 3: 전류 폭풍 — 아레나 전체 전류 패턴
  약점: 전류 안전 경로(사인파 패턴) 관찰 → 타이밍 접근
```

#### 보스 5: 아비소스 핵심 (바이옴 5 — 최종 보스)
```
Phase 1: 고대 기계 — 룬 레이저 격자 + 중력 역전
  약점: 수압 실드(해금 시)로 고수압 경로 돌파 → 룬 비활성화 퍼즐
Phase 2: 차원 균열 — 이전 바이옴 함정 랜덤 소환
  약점: 모든 도구를 상황별 활용 (복합 공략)
Phase 3: 문명의 심장 — 4개 코어를 순서대로 비활성화
  약점: 각 코어에 특정 도구 필요 (음파→전기→조명→실드)
Phase 4: 최후 — 코어 4개 동시 재활성, 시간 제한 30초
  약점: 역순 비활성화 (실드→조명→전기→음파) 빠른 실행
```

**bossRewardGiven 플래그 패턴** (F17): 보스 보상(도구 해금)은 `bossRewardGiven[bossId] = true` 후 중복 지급 방지.

## §8. 난이도 시스템

### §8.1 3구간 밸런스 테이블

| 구간 | 바이옴 | 산소 시간 | 함정 밀도 | 함정 속도 | 수압 데미지 | 해류 세기 |
|------|--------|----------|----------|----------|-----------|----------|
| 초반 | 1~2 | 90초/60초 | 3~5/화면 | ×1.0 | 0/낮음 | 약 (0.2) |
| 중반 | 3 | 50초 | 5~7/화면 | ×1.3 | 중간 (2/초) | 중 (0.5) |
| 후반 | 4~5 | 40초/35초 | 7~10/화면 | ×1.6 | 높음 (4/초) | 강 (0.8) |
| 히든 | H-1~3 | 30초 | 10+/화면 | ×2.0 | 극 (6/초) | 난류 |

### §8.2 밸런스 산식 및 가정

**생존 시간 산식**:
```
survivalTime = oxygenTime / (1 + pressureDrain + environmentMod)
필요 클리어 시간 = stageLength / (swimSpeed * depthFactor)
밸런스 조건: survivalTime > 필요 클리어 시간 * 1.3 (30% 여유)
```

**가정** (F25):
- 플레이어 회피율: 초반 70%, 중반 55%, 후반 40%
- 산소 버블 수집률: 60%
- 도구 사용 효율: 50% (쿨다운 중 미사용 비율)

**DDA 폴백** (3연속 사망 시):
1. 함정 30% 비활성화 (시각적으로 dim 처리)
2. 산소 +20% 보너스
3. 5연속 사망 시 추가: 수압 데미지 50% 감소
4. 10연속 사망 시 추가: "유령 경로" 힌트 표시 (안전 경로 점선)

### §8.3 극단 빌드 밸런스 검증 (부록 A 참조)

| 빌드 | 업그레이드 집중 | 바이옴 5-3 생존 시간 | 클리어 가능 여부 |
|------|--------------|-------------------|---------------|
| 내압 올인 | 내압 Lv5, 속도 Lv1, 감지 Lv1 | 52초 (수압 면역) | ✅ 필요 45초 |
| 속도 올인 | 내압 Lv1, 속도 Lv5, 감지 Lv1 | 28초 (빠르지만 수압에 약함) | ⚠️ 경계선 (필요 25초, 회피율 의존) |
| 감지 올인 | 내압 Lv1, 속도 Lv1, 감지 Lv5 | 35초 (모든 함정 사전 인지) | ✅ 필요 35초 (함정 100% 인지) |

### §9.1 잠수복 업그레이드 트리 (영구 — 생존력 축)

업그레이드 포인트: 보스 격파(5pt) + 스테이지 클리어(1pt) + 문명 단서(2pt)

        [내압 강화]          [추진력 강화]         [감지 강화]
  Lv1: 수압 내구 +20%    Lv1: 이동속도 +10%    Lv1: 함정 경고 범위 +1
  Lv2: 수압 내구 +40%    Lv2: 이동속도 +20%    Lv2: 함정 경고 범위 +2
  Lv3: 수압 내구 +70%    Lv3: 부스트 시간 +1초   Lv3: 숨겨진 아이템 표시
  Lv4: 수압 면역 3초쿨   Lv4: 급강하 무적 0.5초  Lv4: 보스 약점 하이라이트
  Lv5: 수압 존 감속 면역  Lv5: 벽타기 + 천장 이동  Lv5: 미니맵 상시 표시
```

비용: Lv1(3pt), Lv2(5pt), Lv3(8pt), Lv4(12pt), Lv5(18pt)
총 필요: 46pt × 3갈래 = 138pt (전체 클리어 기준 약 80pt 획득 → 1~2갈래 집중 전략)

### §9.2 도구 언락 (런별 — 퍼즐 접근성 축) (F24)
도구는 보스 격파로 해금되며 **영구 보유**. 런별 변동 요소는 없음 — 영구 진행(업그레이드)과 도구 언락이 **다른 축(생존력 vs 퍼즐 접근성)**을 강화하여 상호 보완 (F24).

### §9.3 문명 단서 도감
15개 문명 단서를 수집하면 아비소스 문명의 스토리 단편이 해금:
- 바이옴별 3개 × 5바이옴 = 15개
- 각 단서 위치는 고정이나 접근 경로의 함정이 변동 (SeededRNG)
- 전부 수집 시 히든 H-3 해금 + 진 엔딩

## §10. 레벨 생성 시스템

### §10.1 하이브리드 생성 전략 (F26)
- **기본 레이아웃**: 각 스테이지의 지형 타일맵은 **사전 정의** (15+5+3=23개 수동 설계)
- **변형 요소** (SeededRNG 40~60% 랜덤화):
  - 함정 배치: 지정 후보 위치 중 40~60% 활성화
  - 산소 버블 위치: 후보 위치 중 랜덤 선택
  - 해류 방향/세기: sinewave 위상 랜덤
  - 적대 생물 경로: waypoint 기반 경로 변형

### §10.2 도달 가능성 검증 (BFS)
레벨 생성 후 반드시 BFS 검증:
```
1. 시작 위치 → 탈출구 경로 존재 확인
2. 문명 단서 위치 → 접근 가능 확인
3. 산소 버블 최소 1개가 경로상에 존재 확인 (바이옴 3+ 필수)
4. 검증 실패 시 → 함정 1개씩 비활성화하며 재검증 (최대 3회)
5. 3회 실패 시 → 기본 레이아웃 원본 사용 (함정 변형 0%)
```

### §10.3 시드 시스템
```
stageSeed = baseSeed XOR (biome * 1000 + stage * 100 + attemptCount)
```
- `baseSeed`: 게임 시작 시 `Date.now()` 기반 생성
- `attemptCount`: 해당 스테이지 시도 횟수 (리트라이마다 변형)
- 타임어택 모드: 고정 시드로 공정한 경쟁

### §10.4 도구 해금 순서 → 바이옴 접근 가능성 맵 (F27)

| 바이옴 | 필수 도구 | 해금 조건 | 접근 가능 시점 |
|--------|----------|----------|-------------|
| 1. 산호 정원 | 없음 | 게임 시작 | 즉시 |
| 2. 심해 동굴 | 없음 (음파탐지 권장) | 바이옴 1 클리어 | 보스 1 격파 후 |
| 3. 열수 분출구 | 음파탐지 (숨겨진 우회로) | 바이옴 2 클리어 | 보스 2 격파 후 |
| 4. 해구 심연 | 전기충격 + 조명탄 | 바이옴 3 클리어 | 보스 3 격파 후 |
| 5. 아비소스 유적 | 수압 실드 | 바이옴 4 클리어 | 보스 4 격파 후 |
| 히든 H-1~H-3 | 전도구 + 특수 해금 조건 | §2 참조 | 조건부 |

**검증**: 각 바이옴 보스는 해당 시점에서 보유 가능한 도구만으로 격파 가능해야 함.
- 보스 1: 도구 없이 격파 가능 (순수 관찰+회피)
- 보스 2: 음파탐지 없이도 격파 가능 (관찰 난이도 상승)
- 보스 3: 전기충격 필수 (냉각 장치 활성화)
- 보스 4: 조명탄 필수 (어둠 속 여왕 위치 탐지)
- 보스 5: 4도구 모두 필수 (Phase 3 코어 비활성화)

### A.1 내압 올인 빌드
```

- 수압 데미지: 면역 (Lv5 특성)
- 이동 속도: baseSpeed × 1.1 × 0.4(depth) = 0.44 base
  → Lv5 면역으로 depthFactor 무시 = 1.1 base
- 스테이지 길이: 기준 45초 (1.0 speed 기준)
- 실제 필요: 45 / 1.1 = 40.9초
- 산소 35초 < 필요 40.9초 → ❌?
  → 산소 버블 2개 (각 +10초) = 55초 유효
  → 55 - 40.9 = 14.1초 여유 ✅
```

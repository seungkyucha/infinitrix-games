## 1. 게임 개요 및 핵심 재미 요소

**잉크 메이든**은 잉크로 그려진 세계에 갇힌 소녀 '마리'가 잃어버린 붓의 능력을 하나씩 되찾으며, 봉인된 영역을 해금하고 세계를 물들인 어둠의 근원을 쓰러뜨리는 **횡스크롤 메트로배니아**이다.

### 핵심 재미 3요소

1. **탐험의 보상감**: "저 문은 아직 못 열어…" → 능력 획득 후 되돌아오면 새 길 발견 → 도파민 루프
2. **능력 성장의 쾌감**: 대시→이중점프→벽타기 순으로 이동 수단이 확장. 이전에 갈 수 없던 곳이 열리는 순간의 "아하!" 모먼트
3. **보스전 긴장감**: 보스 HP 50% 기준 2페이즈 전환. 패턴을 읽고 피하며 공격 타이밍을 찾는 액션의 재미

### Cycle 3 "아쉬웠던 점" 반영

| C3 아쉬운 점 | 이번 사이클 해결 방안 |
|---|---|
| 3차 리뷰까지 소요 (셀프 QA 부재) | 체크리스트 #14: 풀 플로우 셀프 QA 의무화 |

| 이벤트 다양성 부족 | 맵 곳곳에 비밀 방 + 보상 다양화 (HP 업, 잉크 업, 비밀 통로) |

## 2. 게임 규칙 및 목표

### 승리 조건
- 3개 영역(잉크 숲 → 수묵 동굴 → 먹물 성)의 보스를 모두 처치하고, 최종 보스 "어둠의 붓"을 쓰러뜨리면 클리어

### 패배 조건
- HP가 0이 되면 게임오버 → 마지막 세이브 포인트(잉크 우물)에서 부활 선택 또는 타이틀로 복귀

### 게임 흐름
```
타이틀 → 난이도 선택 → [잉크 숲] → 대시 획득 → [수묵 동굴] → 이중점프 획득
→ [먹물 성] → 벽타기 획득 → [어둠의 방] → 최종 보스 → 엔딩 → 등급 화면
```

### 맵 구조

총 **4개 영역**, 타일 기반 맵 (화면 단위 스크롤):

| 영역 | 타일 수 | 보스 | 획득 능력 | 잠금 해제 조건 |
|------|---------|------|-----------|---------------|
| 잉크 숲 (Ink Forest) | 8×6 화면 | 잉크 늑대 | 대시 | 시작 영역 |
| 수묵 동굴 (Sumi Cave) | 6×5 화면 | 수묵 거미 | 이중점프 | 대시로 부서지는 벽 통과 |
| 먹물 성 (Ink Castle) | 7×5 화면 | 먹물 기사 | 벽타기 | 이중점프로 높은 절벽 도달 |
| 어둠의 방 (Dark Chamber) | 3×3 화면 | 어둠의 붓 (최종) | — | 벽타기로 봉인된 문 해제 |

> ⚠️ 단일 HTML 스코프 관리: 맵은 JSON 데이터로 선언적 정의. 각 화면은 16×10 타일(타일 60×64px). 전체 약 24개 화면 = 관리 가능한 규모.

## 4. 시각적 스타일 가이드

### 색상 팔레트
| 역할 | 색상 | HEX |
|------|------|-----|
| 잉크 검정 (주선/캐릭터 아웃라인) | 짙은 먹색 | `#1a1a2e` |
| 종이 배경 (기본 바탕) | 따뜻한 아이보리 | `#f5f0e1` |
| 수채 파랑 (능력/UI 하이라이트) | 남색 수채 | `#4a7fb5` |
| 수채 빨강 (적/위험/HP) | 주홍 수채 | `#c94c4c` |
| 수채 초록 (치유/세이브 포인트) | 비취 수채 | `#6b9e6b` |
| 수채 보라 (보스/마법) | 자주 수채 | `#7b5ea7` |
| 잉크 금색 (점수/아이템) | 오커 금 | `#d4a847` |

### 배경
- **잉크 숲**: 종이 텍스처 위에 나무와 덩굴이 잉크 선화로 그려짐. 수채 초록/파랑 번짐 효과
- **수묵 동굴**: 짙은 먹색 배경 + 수묵 얼룩. 스토킹라이트 동굴 느낌
- **먹물 성**: 날카로운 잉크 선의 고딕 건축. 보라+빨강 수채 악센트
- **어둠의 방**: 거의 전부 검정 + 캐릭터 주변만 종이 텍스처가 보임

### 오브젝트 형태
- **플레이어 (마리)**: 잉크 선화 소녀, 큰 붓을 들고 있음. 수채 파랑 머리카락
- **적**: 잉크 얼룩에서 태어난 형상. 번진 먹 같은 실루엣 + 빨간 눈
- **아이템**: 잉크 방울(HP 회복), 금빛 잉크 조각(점수)
- **지형**: 종이 접힘선 같은 플랫폼, 잉크 번짐 같은 위험 지대

## 4.5. 아트 디렉션 (Art Direction)

### 아트 스타일 키워드
**"Hand-drawn 2D art: visible ink lines, watercolor-like fills, slight paper texture (Hollow Knight / Cuphead feel)"**

### 핵심 원칙
1. **잉크 선화 우선**: 모든 오브젝트는 명확한 검정 잉크 아웃라인으로 정의
2. **수채화 채움**: 내부는 수채화 번짐 느낌의 색감. 완벽한 채색이 아니라 약간의 번짐/얼룩
3. **종이 텍스처**: 배경에 미세한 종이 결 텍스처가 깔려 "스케치북에 그린" 느낌
4. **제한된 팔레트**: 7색 기본 팔레트 엄수. 영역별로 주요 수채색 1~2개만 강조

### 레퍼런스 게임
- **Hollow Knight**: 잉크 선화 캐릭터 + 분위기 있는 배경 레이어링
- **Cuphead**: 핸드드로우 애니메이션의 매력적 질감 (단, Cuphead보다 수채화 톤이 더 부드러움)

### 스타일 일관성 규칙
- 모든 에셋(캐릭터/배경/UI/썸네일)에 동일한 잉크 선화 + 수채화 스타일 적용
- 잉크 선 굵기: 캐릭터 2~3px, 배경 1~2px, UI 1px (960×640 기준)
- 수채 번짐: 캐릭터는 깔끔하게, 배경은 더 흐릿하게 → 시각적 계층 분리

## 5. 핵심 게임 루프 (프레임 기준 로직 흐름)

게임은 60fps 대상으로 설계. `dt` (delta time) 기반 프레임 독립적 로직.

### 매 프레임 업데이트 순서

```
1. 입력 처리 (Input)
   ├─ 키보드/터치/마우스 상태 읽기
   ├─ 씬 전환 직후면 200ms 입력 무시 (inputDelay)
   └─ 능력 보유 여부에 따라 대시/이중점프/벽타기 입력 허용

2. 물리 업데이트 (Physics)
   ├─ 중력 적용: vel.y += GRAVITY * dt
   ├─ 이동 적용: pos += vel * dt
   ├─ 타일맵 충돌 검사 (AABB)
   ├─ 벽타기 상태면 중력 감소 (vel.y *= 0.3)
   └─ 대시 중이면 무적 + 고정 속도

3. 게임 로직 (Logic)
   ├─ 플레이어 공격 판정 → 적 피격 처리
   ├─ 적 AI 업데이트 (순찰/추적/공격)
   ├─ 아이템 수집 판정
   ├─ 화면 전환 트리거 (화면 경계 도달)
   └─ 세이브 포인트 접촉 판정

4. 카메라 업데이트 (Camera)
   ├─ 목표: 플레이어 위치 + 이동 방향 오프셋
   └─ lerp(0.1) 부드러운 추적 (C2 검증 패턴)

5. 렌더링 (Render)
   ├─ 배경 레이어 (원경 → 중경 → 근경, 패럴렉스)
   ├─ 타일맵 렌더링 (뷰포트 내 타일만)
   ├─ 엔티티 렌더링 (적 → 아이템 → 플레이어 순)
   ├─ 이펙트/파티클 렌더링
   └─ UI 오버레이 (HP바, 능력 아이콘, 미니맵)
```

### 씬 구조

| 씬 | 설명 |
|----|------|
| TITLE | 타이틀 화면 + 난이도 선택 |
| GAMEPLAY | 메인 게임 플레이 (맵 탐험 + 전투) |
| PAUSE | 일시정지 + 맵 보기 |
| BOSS | 보스전 (전용 방) |
| ABILITY | 능력 획득 연출 |
| GAMEOVER | 게임오버 → 부활/타이틀 선택 |
| ENDING | 클리어 엔딩 + 등급 표시 |

> ⚠️ 모든 씬의 enter()에서 해당 씬의 UI 버튼을 반드시 빌드 (C3 교훈).
> ⚠️ enter(data) 패턴으로 씬 간 데이터 전달 — 모듈 레벨 변수에 저장.

## 6. 난이도 시스템

### 3단계 난이도 선택 (타이틀 화면)

| 항목 | 쉬움 (수채화) | 보통 (먹물) | 어려움 (진묵) |
|------|-------------|-----------|-------------|
| 플레이어 HP | 8 | 5 | 3 |
| 적 HP 배율 | ×0.7 | ×1.0 | ×1.5 |
| 적 공격력 배율 | ×0.7 | ×1.0 | ×1.3 |
| 세이브 포인트 간격 | 2~3 화면마다 | 4~5 화면마다 | 영역당 1개만 |
| 무적 시간 (피격 후) | 1.5초 | 1.0초 | 0.5초 |
| 잉크 방울 드롭률 | 40% | 20% | 10% |

### 진행에 따른 동적 난이도

- **영역 1 (잉크 숲)**: 기본 적만 등장. 느린 공격 패턴. 튜토리얼 역할
- **영역 2 (수묵 동굴)**: 원거리 적 추가. 환경 트랩 (잉크 가시) 등장
- **영역 3 (먹물 성)**: 빠른 적 + 공중 적 추가. 복합 트랩
- **최종 보스방**: 모든 능력을 활용해야 하는 복합 패턴

## 9. 능력 시스템 상세

### 능력 목록

| 능력 | 획득 위치 | 입력 | 효과 | 맵 해금 |
|------|----------|------|------|---------|
| **대시** | 잉크 숲 보스 후 | Shift/Z | 이동 방향으로 빠르게 돌진 (0.2초간 무적, 쿨타임 1초) | 부서지는 벽 통과 |
| **이중점프** | 수묵 동굴 보스 후 | 공중 ↑/W | 공중에서 한 번 더 점프 (점프력 100%) | 높은 절벽 도달 |
| **벽타기** | 먹물 성 보스 후 | 벽 접촉 + ↑ | 벽에 달라붙어 위로 이동 (속도: 일반 이동의 60%) | 봉인된 수직 통로 |

### 능력 획득 연출

1. 보스 처치 → 보스 잉크가 능력 오브로 응축
2. 오브 접촉 → ABILITY 씬 전환 (200ms 입력 무시)
3. 능력 이름 + 조작법 표시 (3초)
4. 획득 이펙트(effect-ability-get) 재생
5. GAMEPLAY 씬 복귀 → 새 능력 즉시 사용 가능

## 10. 적 AI 패턴

### 일반 적

| 적 | 행동 패턴 | HP | 공격력 | 등장 영역 |
|----|----------|-----|--------|----------|
| 잉크 얼룩 | 좌우 순찰 → 플레이어 감지 시 돌진 | 2 | 1 | 잉크 숲 |
| 잉크 박쥐 | 공중 순찰 → 감지 시 급강하 | 1 | 1 | 수묵 동굴 |
| 잉크 거미 | 천장 고정 → 감지 시 거미줄 낙하 | 2 | 1 | 수묵 동굴 |
| 먹물 근위병 | 구역 순찰 → 감지 시 검 공격 (2연타) | 4 | 2 | 먹물 성 |

### 보스 패턴

**잉크 늑대 (잉크 숲)**
- 페이즈 1: 돌진(3초 쿨) → 할퀴기(근접) → 울부짖기(전방 충격파)
- 페이즈 2 (HP≤50%): 돌진이 2연속 → 잉크 분출(바닥 3곳에 잉크 웅덩이 생성)

**수묵 거미 (수묵 동굴)**
- 페이즈 1: 천장에서 잉크 방울 낙하(3방향) → 거미줄 내려오기 → 찍기
- 페이즈 2 (HP≤50%): 바닥 이동으로 전환 → 돌진+칼날 다리 휘두르기 → 잉크 안개(시야 제한)

**먹물 기사 (먹물 성)**
- 페이즈 1: 검 내려치기 → 횡베기 → 방패 돌진
- 페이즈 2 (HP≤50%): 검에 잉크 불꽃 → 원거리 잉크 파동 추가 → 망토 촉수 공격

**어둠의 붓 (최종)**
- 페이즈 1: 잉크 소용돌이(화면 절반 장악) → 붓 찍기(낙하 공격) → 잉크 파편 발사
- 페이즈 2 (HP≤50%): 그림자 마리로 변신 → 플레이어 기술 모방(대시/점프/공격) → 모든 능력 활용 필요

## 12. 사운드 디자인 (Web Audio 톤 합성)

외부 오디오 파일 없이 Web Audio API로 실시간 합성 (C1~C3 검증 패턴).

| 이벤트 | 사운드 설계 |
|--------|-----------|
| 붓 공격 | 짧은 스윕 (100Hz→800Hz, 0.1초) + 잔향 |
| 점프 | 사인파 상승 (200Hz→400Hz, 0.15초) |
| 대시 | 화이트 노이즈 버스트 (0.2초) + 저음 붐 |
| 피격 | 노이즈 + 저주파 진동 (0.3초) |
| 적 사망 | 하강 톤 (400Hz→50Hz, 0.4초) + 수분 느낌 잔향 |
| 아이템 수집 | 맑은 종소리 (800Hz, 0.2초) |
| 능력 획득 | 상승 아르페지오 (C4→E4→G4→C5, 1초) |
| 보스 등장 | 깊은 드론 (50Hz, 2초 페이드인) + 불협화음 |
| 보스 2페이즈 | 강렬한 임팩트 (60Hz, 0.5초) + 역잔향 |
| 세이브 | 부드러운 코드 (C-E-G, 0.5초) |

## 15. 셀프 QA 프로토콜 (필수)

> ⚠️ C1~C3 3사이클 연속 3차 리뷰. 이번 사이클에서 반드시 끊는다.

### 코드 제출 전 필수 플로우 테스트

```
1. 로드 → 타이틀 화면 정상 표시 확인
2. 난이도 선택 → GAMEPLAY 진입 확인
3. 잉크 숲 탐험 → 일반 적 전투 → 세이브 포인트 사용
4. 잉크 숲 보스전 → 2페이즈 전환 확인 → 처치
5. 대시 능력 획득 연출 → 부서지는 벽 통과
6. 수묵 동굴 진입 → 탐험 → 보스전 → 이중점프 획득
7. 먹물 성 진입 → 보스전 → 벽타기 획득
8. 어둠의 방 → 최종 보스전 → 클리어 → 등급 화면
9. 타이틀로 복귀 → 재시작 1회 → JS 에러 0건 확인
10. 3회 연속 재시작 → 능력 상태/HP/점수 완전 리셋 확인
```

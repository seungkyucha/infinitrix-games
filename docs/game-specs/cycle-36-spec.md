# 메카 개리슨 (Mecha Garrison) — 사이클 #36 게임 기획서

> **1페이지 요약**: 외계 기계 군단 "볼텍스"의 침공에 맞서 인류 최후의 메카 기지를 방어하는 **타워디펜스 로그라이트**. 5개 구역(도시 외곽→산업 지구→에너지 코어→지하 벙커→최종 사령부) × 3웨이브 + 구역 보스 5종 + 히든 스테이지 2개 = **총 17+ 스테이지**. 메카 유닛 5종 배치 + 부품 조합(3부품→무장) + 3갈래 × 5단계 영구 업그레이드 트리 + 환경 전략(주간/야간 + 산성비/EMP) + DDA 4단계 보정. SeededRNG 기반 적 웨이브·부품 드롭 랜덤화로 높은 리플레이 가치. **arcade+strategy 조합으로 8사이클 최장 미사용(#25 이후) 해소 + 메카 미사용 테마 소진.** 2026 HTML5 최고 트렌드 "TD+Roguelite" 직접 대응. 10초 안에 이해, 10시간 파고들기.

> **MVP 경계**: Phase 1(핵심 루프: 배치→웨이브 방어→보상→강화, 구역 1~3 + 보스 3체 + 메카 3종 + 업그레이드 Lv1~3 + 부품 8종 + DDA) → Phase 2(구역 4~5 + 보스 2체 + 히든 2스테이지 + 메카 5종 완성 + 업그레이드 Lv4~5 + 부품 15종 + 날씨/시간대 + 전체 내러티브). **Phase 1만으로도 완전한 타워디펜스 로그라이트 경험 제공 필수.**

## §1. 게임 개요 및 핵심 재미 요소

### 핵심 컨셉
"최후의 기지를 지켜라 — 매 출격이 더 강해진다" — 외계 기계 군단 볼텍스의 침공에 맞서 메카 유닛을 전략적으로 배치하고 강화하는 **타워디펜스 로그라이트**.

### 핵심 재미 3축
1. **전략적 배치**: 5종 메카 유닛의 사거리·특성을 고려한 그리드 배치. "어디에 무엇을 놓을 것인가?"
2. **부품 조합의 쾌감**: 웨이브 클리어 후 랜덤 부품 3개 중 1개 선택 → 메카 무장 커스터마이즈. 매 런마다 다른 빌드
3. **영구 성장의 동기**: 런 종료 시 획득한 코어로 영구 업그레이드. "다음 런에서는 더 멀리 갈 수 있다"

### 시장 포지셔닝
- Poki Top 10에 순수 타워디펜스 0개 → **블루오션**
- itch.io TD+Roguelite 최고 성장 트렌드 → 대형 포털 침투 임박
- arcade 성격(빠른 웨이브, 직관적 배치) + strategy 깊이(업그레이드 트리, 부품 조합) = 폭넓은 어필

### 차별화 (기존 arcade+strategy 2개 대비)
- chrono-siege: 시간 조작 공성 → **실시간 TD + 로그라이트 메타 진행** (완전 별개 메카닉)
- void-architect: 건축 퍼즐 → **유닛 배치 + 웨이브 서바이벌** (완전 별개 메카닉)

## §2. 게임 규칙 및 목표

### 승리 조건
- 5구역 × 3웨이브 + 구역 보스 5종을 모두 돌파하여 최종 사령부를 방어
- 히든 스테이지 2개 클리어 시 트루 엔딩

### 패배 조건
- 기지 HP(코어)가 0이 되면 런 종료
- 런 종료 시 획득한 코어 포인트로 영구 업그레이드 (로그라이트)

### 핵심 규칙
1. **배치 페이즈** (PLACEMENT): 보유 에너지로 메카 유닛을 그리드에 배치/업그레이드. 시간 제한 없음
2. **전투 페이즈** (WAVE): 적이 경로를 따라 이동. 메카가 자동 공격. 플레이어는 스킬 발동만 가능
3. **보상 페이즈** (REWARD): 웨이브 클리어 후 부품 3개 중 1개 선택 (로그라이크 선택지)
4. **보스 페이즈** (BOSS): 구역 마지막 웨이브 — 대형 보스 출현, 패턴 공략 필요
5. **기지 귀환** (HUB): 런 종료(클리어/사망) 시 코어 포인트로 영구 업그레이드

### BFS 경로 시스템
- 적은 출입구→기지까지 BFS 최단 경로로 이동
- 메카 배치 시 즉시 BFS 재계산 → **경로 미존재 시 배치 거부** (§10.2 Cycle 26 교훈)
- 경로 유도(마징) = 핵심 전략

### §8.2 DPS/EHP 밸런스 산식

**가정** (Cycle 24 교훈 — 가정을 명시):
- 플레이어 메카 배치 효율: 70% (최적 대비)
- 평균 메카 레벨: 구역 1=Lv1, 구역 3=Lv2, 구역 5=Lv2.5
- 부품 보너스: 커먼 평균 12.5%, 구역당 누적

**산식**:
```
팀DPS = Σ(메카_공격력 × 레벨배율 × 부품배율) / 공속
웨이브EHP = Σ(적_HP × HP배율) / 적_수
클리어시간 = 웨이브EHP / 팀DPS
목표: 30초 ≤ 클리어시간 ≤ 120초
```

**가정 오류 시 DDA 폴백** (Cycle 24 교훈):
- 배치 효율이 50% 미만으로 추정되면 → DDA 레벨 1 자동 적용

## §10. 레벨 생성 시스템

### §10.1 그리드 맵 구조
- 구역별 맵: 12×8 (PC) / 8×6 (모바일) 그리드
- 입구(좌/상) → 출구(우하 = 기지) BFS 경로
- 장애물 타일: 구역별 20~35% 사전 배치

### §10.2 하이브리드 프로시저럴 생성 (F26, Cycle 33 교훈)
1. **기본 레이아웃**: 구역별 3개 사전 정의 맵 템플릿 (장애물 위치 고정)
2. **SeededRNG 변형** (40~60%):
   - 장애물 ±2타일 이동
   - 적 스폰 구성 변형
   - 부품 드롭 테이블 셔플
   - 환경(시간대/날씨) 랜덤 할당
3. **BFS 도달 가능성 검증**: 변형 후 즉시 BFS 실행 → 경로 미존재 시 변형 롤백
4. **배치 시 실시간 BFS**: 메카 배치 시 즉시 BFS 재계산 → 경로 차단 시 배치 거부 (Cycle 26 교훈)
5. **보스 약점 위치**: BFS 경로에서 최소 1셀 이격 보장 (Cycle 26 교훈)

### §10.3 BFS 경로 알고리즘
```
function findPath(grid, start, end):
  queue = [start]
  visited = Set(start)
  parent = Map()
  while queue.length > 0:
    cell = queue.shift()
    if cell === end: return reconstructPath(parent, end)
    for neighbor in getNeighbors(cell, grid):
      if !visited.has(neighbor) && !isBlocked(neighbor):
        visited.add(neighbor)
        parent.set(neighbor, cell)
        queue.push(neighbor)
  return null  // 경로 없음 → 배치 거부
```

## 부록 A. 극단 빌드 밸런스 검증

### 빌드 1: 화력 올인 (블래스터 ×6 Lv3 + 에픽 공격력 부품 3개)
```
기본 DPS = 6 × (10 × 1.96) / 1.0 = 117.6/s
부품 보너스 = DPS 캡 200% → 117.6 × 2.0 = 235.2/s (캡)
구역 5 웨이브 EHP = 35 × (80 × 2.5) = 7,000
클리어 시간 = 7,000 / 235.2 ≈ 30초 ✅ (범위 내)
```

### 빌드 2: 탱커 (실드 ×3 + 방어 Lv5 + DDA Lv0)
```
기지 HP = 500 + 100(Lv1) + 방어 Lv5 보너스 = 800
적 DPS (구역 5) = ~15/s (적이 기지 도달 시)
생존 시간 = 800 / 15 ≈ 53초
필요 아군 DPS = 7,000 / 53 ≈ 132/s → 블래스터 Lv2 ×4로 달성 가능 ✅
```

### 빌드 3: 유틸 특화 (유틸 Lv5 + 레일건 ×2 + 에너지 부품)
```
에너지 수입 = 기본 5E/킬 + 유틸 Lv3 판매환불 20% → 실질 8E/킬
레일건 DPS = 2 × (80 × 1.4) / 5.0 = 44.8/s (관통으로 ×2~3 히트)
실효 DPS = 44.8 × 2.5(평균 관통) = 112/s
구역 5 클리어 시간 = 7,000 / 112 ≈ 63초 ✅ (범위 내)
```

**결론**: 3개 극단 빌드 모두 30~120초 클리어 시간 범위 내. 밸런스 유효.

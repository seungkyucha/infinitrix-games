## 1. 게임 개요 및 핵심 재미 요소

**셀 아레나**는 유닛을 드래프트하고 전략적으로 배치하여 자동 전투를 관전하는 로그라이크 오토 배틀러입니다.

### 배경 스토리
셀 셰이드 세계 "아레나"에서 다섯 부족의 전사들이 패권을 다투고 있습니다. 플레이어는 지휘관으로서 매 라운드 상점에서 용병을 고용하고, 전투 그리드에 배치하여 적 진영과 자동 전투를 벌입니다. 부족과 직업 시너지를 조합해 최강의 팀을 구축하고, 15라운드를 생존하여 최종 보스를 격파하세요.

### 핵심 재미 요소
1. **"이 조합이면?"** — 5부족 × 3직업 = 15유닛, 10종 시너지 조합의 전략적 깊이. TFT/Super Auto Pets에서 검증된 "시너지 발견의 쾌감"
2. **"배치가 승부를 가른다"** — 5×4 그리드에서 탱커 전열/딜러 후열/힐러 안전 위치 등 포지셔닝 전략
3. **"한 판 더"** — 라운드마다 달라지는 상점 라인업 + 리롤 도박 + 메타 업그레이드로 반복 플레이 유도
4. **"관전의 긴장감"** — 배치 완료 후 자동 전투를 지켜보며 내 전략이 통하는지 확인하는 짜릿함
5. **보스 2페이즈** — R5/R10/R15 보스전에서 HP 50% 이하 시 패턴 변환, 긴장감 극대화

### 레퍼런스
- **Super Auto Pets**: 단순한 UI + 깊은 시너지 조합 = 브라우저 오토 배틀러의 교과서
- **Teamfight Tactics (TFT)**: 부족/직업 이중 시너지 + 그리드 배치 + 경제 관리
- **Thronefall**: 미니멀 오토 배틀 + 배치 전략의 인디 히트작

## 2. 게임 규칙 및 목표

### 승리 조건
- 15라운드를 생존하고 최종 보스(R15)를 격파

### 패배 조건
- 지휘관 HP가 0 이하로 떨어지면 게임 오버 (초기 HP: Easy 30 / Normal 20 / Hard 15)
- 전투 패배 시 남은 적 유닛 수만큼 HP 감소

### 라운드 구조
```
[상점 페이즈] → [배치 페이즈] → [전투 페이즈] → [결과 페이즈] → 다음 라운드
```

1. **상점 페이즈** (시간 무제한): 골드로 유닛 구매(3골드), 리롤(1골드), 유닛 판매(1골드 환급), 유닛 합성(동일 유닛 3개 → ★2, ★2 3개 → ★3)
2. **배치 페이즈**: 5×4 그리드의 아군 영역(하단 5×2)에 유닛 드래그 배치. 최대 배치 수: 라운드에 따라 3→5개 (R1: 3, R3: 4, R6: 5)
3. **전투 페이즈**: 자동 전투. 유닛들이 가장 가까운 적을 향해 이동+공격. 한쪽이 전멸하면 종료
4. **결과 페이즈**: 승리 시 보너스 골드(+1~3), 패배 시 HP 감소, 무조건 기본 골드(+5) + 이자(10골드당 +1, 최대 +5)

### 유닛 시스템
- **5부족**: 불꽃(Flame), 얼음(Frost), 자연(Nature), 암흑(Shadow), 빛(Light)
- **3직업**: 전사(Warrior), 궁수(Ranger), 마법사(Mage)
- **총 15유닛**: 각 부족 × 각 직업 = 1유닛
- **합성**: 동일 유닛 3개 → ★2 (스탯 ×1.8), ★2 3개 → ★3 (스탯 ×3.0)

### 시너지 시스템 (10종)

**부족 시너지** (같은 부족 유닛 수):
| 부족 | 2개 | 3개 |
|------|-----|-----|
| 불꽃 | 공격력 +15% | 공격력 +30%, 공격 시 화염 데미지 추가 |
| 얼음 | 방어력 +20% | 방어력 +40%, 피격 시 적 공속 감소 |
| 자연 | 매 5초 HP 회복 5% | 매 3초 HP 회복 8%, 사망 시 1회 부활 |
| 암흑 | 치명타 확률 +15% | 치명타 확률 +30%, 치명타 데미지 ×2 |
| 빛 | 아군 전체 공속 +10% | 아군 전체 공속 +25%, 전투 시작 시 실드 |

**직업 시너지** (같은 직업 유닛 수):
| 직업 | 2개 | 3개 |
|------|-----|-----|
| 전사 | 최대 HP +200 | 최대 HP +500, 받는 피해 -10% |
| 궁수 | 사거리 +1칸 | 사거리 +2칸, 2번째 공격마다 2회 타격 |
| 마법사 | 스킬 데미지 +20% | 스킬 데미지 +50%, 스킬 쿨다운 -30% |

## 4.5. 아트 디렉션 (Art Direction)

### 아트 스타일 키워드
**cel-shaded** — Cel-shaded 3D: bold outlines, flat color zones, anime/comic book shading (Jet Set Radio / Borderlands feel).

### 스타일 규칙
- 모든 에셋에 **3px+ 검정 아웃라인** 필수
- 색상은 **2~3톤 플랫 셰이딩** (밝은면/그림자면/리플렉션)
- 안티앨리어싱 허용하되, 아웃라인은 항상 선명하게
- 그라데이션은 최소한으로 — 톤 경계가 명확한 셀 셰이딩
- 부족별 컬러 코딩이 유닛 실루엣만으로도 구별 가능해야 함

### 레퍼런스 게임
1. **Jet Set Radio** — bold outline + 밝고 강렬한 flat color, 셀 셰이드의 교과서
2. **Borderlands 시리즈** — 두꺼운 잉크 아웃라인 + 만화적 셰이딩, 전투 이펙트 스타일

## 5. 핵심 게임 루프 (초당 프레임 기준 로직 흐름)

### 5.1 씬 구조
```
TITLE → DIFFICULTY → SHOP → BATTLE → RESULT → SHOP (반복)
                                                  ↓ (HP ≤ 0)
                                              GAMEOVER
                                                  ↓ (R15 보스 격파)
                                              VICTORY
```

**씬 목록** (8개):
1. **TITLE**: 게임 로고 + 시작/조작법/메타 업그레이드 버튼
2. **DIFFICULTY**: Easy/Normal/Hard 선택
3. **CONTROLS**: 조작법 안내
4. **SHOP**: 상점 + 유닛 배치 (핵심 씬)
5. **BATTLE**: 자동 전투 관전
6. **RESULT**: 라운드 결과 (승/패/보상)
7. **GAMEOVER**: 사망 — 최종 점수 + 메타 업그레이드 골드 적립
8. **VICTORY**: 15라운드 클리어 — 최종 점수 + 보너스

### 5.3 프레임 로직 (60fps 기준)

**SHOP 씬 (매 프레임)**:
1. 입력 처리 → 드래그 상태 업데이트
2. 유닛 드래그 중이면 그리드 스냅 위치 계산
3. 시너지 계산 (배치된 유닛 기준)
4. UI 업데이트 (골드, HP, 시너지 패널, 유닛 스탯)
5. 유닛 idle 애니메이션 프레임 진행

**BATTLE 씬 (매 프레임)**:
1. 전투 타이머 += dt × speedMultiplier
2. 각 유닛: 타겟 탐색 → 이동 or 공격 판정
3. 공격 시: 데미지 계산 (시너지 보너스 적용) → HP 갱신 → 피격 이펙트
4. 사망 유닛: 사망 이펙트 → 풀 반환
5. 승리/패배 판정: 한쪽 전멸 시 RESULT 씬 전환
6. 보스 2페이즈: boss.hp ≤ maxHp × 0.5 → 페이즈 전환 연출 (3초) + 스탯 변경

> ⚠️ resize 이벤트 리스너로 현재 씬 UI 재배치. 그리드/상점/시너지 패널 위치 재계산.

## 10. 튜토리얼 (첫 플레이 가이드)

첫 플레이 감지 (localStorage `celArena_tutorialDone` 확인):

1. **R1 상점**: "상점에서 유닛을 구매하세요! (클릭 또는 1~5키)" → 유닛 구매 시 다음
2. **R1 배치**: "유닛을 그리드에 드래그하여 배치하세요" → 배치 완료 시 다음
3. **R1 전투**: "Space를 눌러 전투를 시작하세요!" → 전투 진행
4. **R2 시작**: "같은 유닛 3개를 합치면 ★2로 강화됩니다" → R 키로 리롤 안내
5. **R3 시너지**: "같은 부족/직업 유닛을 배치하면 시너지 보너스!" → Tab으로 시너지 패널

> 힌트는 화면 상단에 반투명 패널로 표시, 아무 키/탭으로 닫기

## 12. 셀프 QA 체크리스트 (5단계 체크포인트)

### CP1: 유닛 시스템 (에셋 로드 + 유닛 데이터)

- [ ] 유닛 렌더링 (셀 셰이드 스타일 일관성)

### CP2: 상점 + 배치

- [ ] 그리드 드래그 배치 동작 (마우스 + 터치)

- [ ] 골드 계산 정확
- [ ] 모든 버튼 48px+ 확인

### CP3: 전투 시스템

- [ ] 시너지 보너스 계산 정확
- [ ] 보스 2페이즈 전환 연출
- [ ] 승리/패배 판정 + HP 감소
- [ ] 전투 속도 토글 (×1/×2) + 시각 피드백

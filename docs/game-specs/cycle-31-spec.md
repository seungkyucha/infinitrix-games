# 철갑 선봉대 (Ironclad Vanguard) — 사이클 #31 게임 기획서

> **1페이지 요약**: 스팀펑크 도시를 침략한 부패 기계왕의 군단에 맞서, 레지스탕스 수석 엔지니어로서 3종 클럭워크 유닛(스트라이커/거너/엔지니어)을 실시간 지휘하여 5개 구역을 탈환하는 **실시간 전술 액션 로그라이트**. 5구역×3단계 = 15 기본 스테이지 + 히든 존 1 = **총 16 스테이지**. 구역 보스 5체 + 히든 보스 "클럭워크 아르케" = **6체 보스**. 워크샵 업그레이드 3트리(공격/방어/생산), 블루프린트 14종(common 6/rare 5/epic 3) 빌드 다양성, SeededRNG 프로시저럴 구역 변형, 3단 난이도(견습생/기술자/마스터) + DDA 3단계 동적 밸런스, 다국어(한/영). **action+strategy 조합을 2→3개로 강화**하며 플랫폼 최초 스팀펑크 테마를 도입한다.

> **MVP 경계**: Phase 1(핵심 루프: 배치→전투→자원수집→업그레이드→보스, 구역 1~2 + 보스 2체 + 유닛 3종 기본형 + 블루프린트 7종 + 워크샵 기본 트리) → Phase 2(구역 3~5 + 보스 3체 + 히든 보스 + 전체 내러티브 + 유닛 진화 + 블루프린트 7종 추가). **Phase 1만으로도 완전한 게임 경험 제공 필수.**

## §1. 게임 개요 및 핵심 재미 요소

### 1.1 컨셉
철갑 선봉대는 스팀펑크 산업 도시 "코그하임(Cogheim)"을 부패한 기계왕 "페릭스(Ferrix)"의 군단으로부터 탈환하는 **실시간 전술 액션 로그라이트**다. 플레이어는 레지스탕스 수석 엔지니어로서, 3종 클럭워크 유닛(스트라이커/거너/엔지니어)을 배치·지휘하고, 스팀 게이지를 관리하며, 5개 구역의 보스를 격파한다.

핵심 차별점은 "유닛 배치 전략 × 실시간 액션 × 로그라이트 빌드"의 3축 복합 시스템이다. 타워 디펜스의 배치 전략과 액션 RPG의 실시간 전투를 결합하되, 매 런마다 달라지는 블루프린트 선택과 워크샵 영구 업그레이드가 리플레이 가치를 보장한다.

### 1.2 핵심 재미 3축
1. **전술적 배치 (Deploy & Command)**: 3종 유닛의 배치 위치·타이밍·조합이 전투 결과를 결정. "거너 2기를 고지대에 배치하고 스트라이커로 전선을 유지할까, 엔지니어로 수리하며 소모전을 할까?"
2. **실시간 액션의 긴장감 (Fight & React)**: 적 웨이브가 실시간으로 밀려오며, 유닛 재배치·스킬 발동·긴급 수리를 빠르게 판단해야 한다. 보스전에서는 패턴 읽기 + 유닛 포메이션 변환이 핵심.
3. **로그라이트 빌드 성장 (Build & Evolve)**: 보스 클리어 시 3택 블루프린트로 유닛을 강화하고, 매 런 축적되는 기어로 워크샵을 영구 업그레이드. "이번 런은 거너 특화 빌드로, 다음엔 엔지니어 힐러 빌드로."

### 1.3 스토리/내러티브
- **배경**: 증기 기관이 문명을 이끄는 도시 코그하임. 도시의 핵심 에너지원인 "에테르 코어"를 관리하던 수석 기계공 페릭스가 에테르의 부패에 감염되어 기계 군단을 일으킨다. 플레이어는 레지스탕스의 수석 엔지니어로서 5개 구역을 하나씩 탈환해야 한다.
- **구역별 이야기**: 각 구역 진입 시 이전 엔지니어의 작업 일지가 Canvas 연출로 표시된다.
  - **증기 항구(Steam Harbor)**: 1대 엔지니어 "볼트"의 일지 — 봉기의 시작
  - **톱니 공장(Gear Factory)**: 2대 엔지니어 "스프링"의 일지 — 생산 라인 장악
  - **용광로 지대(Furnace District)**: 3대 엔지니어 "앤빌"의 일지 — 화염 속 저항
  - **시계탑 광장(Clocktower Plaza)**: 4대 엔지니어 "펜듈럼"의 일지 — 시간의 역전
  - **에테르 코어(Aether Core)**: 페릭스의 일지 — 부패의 진실
- **스테이지 간 텍스트**: Canvas 연출 2~3줄 + 엔지니어 포트레이트 (총 15회 + 히든 2회)
- **엔딩 분기**:
  - 5존 탈환 → **"코그하임 해방"** 일반 엔딩: 에테르 코어를 정화하고 도시를 되찾는다.
  - 히든 보스 "클럭워크 아르케(Clockwork Arche)" 격파 → **"에테르의 계승"** 진엔딩: 아르케는 에테르 코어가 만든 도시의 수호자였으며, 플레이어가 새로운 수호자로 계승한다.

## §2. 게임 규칙 및 목표

### 2.1 승리 조건
- **일반 클리어**: 5개 구역의 보스를 모두 처치하고 에테르 코어를 정화
- **진 클리어**: 5보스 처치 후 해금되는 히든 구역에서 클럭워크 아르케 격파

### 2.2 패배 조건
- 지휘관 HP가 0이 되면 현재 런 종료 → 워크샵 영구 업그레이드 유지, 블루프린트·자원은 리셋
- 런 시작 시 지휘관 HP 100 + 워크샵 보너스

### 2.3 핵심 규칙

#### 2.3.1 유닛 시스템
3종 유닛 + 3단계 진화 (진화는 블루프린트로 해금):

| 유닛 | 역할 | 스팀 소모 | 기본 DPS | 사정거리 | 특성 |
|------|------|----------|---------|---------|------|
| 스트라이커(Striker) | 근접 전사 | 20 | 15 | 1.5타일 | 전방 방어력 +30%, 도발 |
| 거너(Gunner) | 원거리 포격 | 30 | 22 | 4.5타일 | 관통 사격, 고지대 보너스 +15% |
| 엔지니어(Engineer) | 지원 수리 | 25 | 8 | 3타일 | 아군 수리(초당 5HP), 바리케이드 설치 |

**진화 트리** (블루프린트로 해금, 각 유닛 3단계):
- 스트라이커: 돌격병 → 철갑 기사 → 증기 챔피언
- 거너: 소총병 → 캐논 마스터 → 에테르 포격수
- 엔지니어: 정비공 → 발명가 → 마스터 아키텍트

#### 2.3.2 자원 시스템
| 자원 | 획득 | 소모 | 런 간 보존 |
|------|------|------|-----------|
| 스팀(Steam) | 시간 경과(초당 3) + 스팀 벤트 파괴 | 유닛 소환, 스킬 사용 | ❌ |
| 기어(Gear) | 적 처치, 상자, 보스 보상 | 워크샵 업그레이드 | ✅ (영구) |
| 블루프린트(Blueprint) | 보스 클리어 시 3택 | 유닛 진화, 특수 능력 | ❌ (런별) |

#### 2.3.3 스팀 게이지 밸런스
- 최대 스팀: 100 (워크샵으로 최대 150까지 확장)
- 자연 회복: 초당 3 (워크샵으로 최대 5까지 확장)
- 유닛 소환 후 쿨다운: 3초 (같은 유닛 타입)
- **극단 빌드 방지**: 스팀 소모 캡 = 최대 스팀의 80%. 동시 소환 불가(쿨다운 중첩 금지)

## §5. 핵심 게임 루프 (초당 프레임 기준 로직 흐름)

### 5.1 초기화 순서 (F12: TDZ 방지)
```
1. CONFIG 상수 선언
2. 전역 상태 객체(G) 선언 + 기본값
3. Canvas DOM 할당
4. TweenManager/ObjectPool/SoundManager/InputManager 인스턴스 생성
5. 이벤트 리스너 등록
6. gameLoop 시작
```

### 5.2 메인 루프 (requestAnimationFrame)
```
function gameLoop(timestamp) {
    const dt = Math.min((timestamp - lastTime) / 1000, 0.05); // 프레임 캡
    lastTime = timestamp;
    gt += dt; // F19: gameTime

    inputManager.update();

    switch(G.state) {
        case 'TITLE':      updateTitle(dt, gt);      break;
        case 'WORKSHOP':   updateWorkshop(dt, gt);   break;
        case 'ZONE_SELECT':updateZoneSelect(dt, gt);  break;
        case 'DEPLOY':     updateDeploy(dt, gt);     break;
        case 'COMBAT':     updateCombat(dt, gt);     break;
        case 'BOSS_INTRO': updateBossIntro(dt, gt);  break;
        case 'BOSS_FIGHT': updateBossFight(dt, gt);  break;
        case 'STAGE_CLEAR':updateStageClear(dt, gt); break;
        case 'ZONE_CLEAR': updateZoneClear(dt, gt);  break;
        case 'GAME_OVER':  updateGameOver(dt, gt);   break;
        case 'VICTORY':    updateVictory(dt, gt);    break;
        case 'PAUSE':      updatePause(dt, gt);      break;
        case 'MODAL':      updateModal(dt, gt);      break;
    }

    tweenManager.update(dt); // F5: 모든 상태에서 tween 업데이트
    soundManager.update(dt);

    render(gt);
    requestAnimationFrame(gameLoop);
}
```

**핵심 규칙**:
- setTimeout 0건 (F4) — 모든 지연은 tween onComplete 사용
- 가드 플래그 필수 (F5) — `if (transitioning) return;`
- 단일 값 갱신 경로 (F14) — 스팀 게이지는 tween OR 직접 대입, 절대 혼용 금지
- SeededRNG 완전 사용 (F18) — Math.random 0건
- TweenManager.clearImmediate() 분리 (F13) — cancelAll과 add 사이 경쟁 조건 방지

## 부록 A: 극단 빌드 검증

### 가정
- 플레이어 유효 DPS = 이론 DPS × 0.6 (회피/재배치 시간 고려)
- 보스 유효 DPS = 보스 ATK × 0.5 (플레이어 방어/회피)

### 극단 빌드 1: 거너 올인 (거너 3기 + 공격 트리 Lv5 + Epic 거너 블루프린트)
- 거너 DPS: 22 × 1.5(워크샵) × 1.5(에픽 블루프린트) = 49.5 → 캡(200%) = 44
- 유닛 3기 총 DPS: 44 × 3 = 132
- 유효 DPS: 132 × 0.6 = 79.2
- 페릭스(HP 1500) 예상 클리어: 1500 / 79.2 = 약 19초 → ✅ 적정 (15~90초 범위)

### 극단 빌드 2: 엔지니어 탱커 (엔지니어 3기 + 방어 트리 Lv5)
- 유닛 DPS: 8 × 3 = 24, 유효: 14.4
- 페릭스 클리어: 1500 / 14.4 = 약 104초 → ⚠️ 길지만, 수리로 생존 가능
- DDA Challenge 적용 시 적 HP +10%: 1650 / 14.4 = 115초 → ⚠️ 경계, DDA Ease로 폴백

### 극단 빌드 3: 혼합 최적 (STR 1 + GUN 1 + ENG 1 + 균등 업그레이드)
- 총 DPS: (15+22+8) × 1.25 = 56.25, 유효: 33.75
- 페릭스 클리어: 1500 / 33.75 = 약 44초 → ✅ 적정

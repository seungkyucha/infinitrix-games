# 엘리멘탈 캐스케이드 (Elemental Cascade) — 사이클 #27 게임 기획서

> **1페이지 요약**: 6원소(화염/물결/대지/바람/빛/암흑) 보석을 매칭하여 마나를 생성하고, 전략적 스펠 시전으로 적을 처치하며 봉인된 던전을 공략하는 **원소 매치-3 로그라이크 RPG**. 8×8 보석 그리드에서 3매치→4매치(십자 폭발)→5매치(전체 폭발)→L/T매치(특수효과)로 콤보를 연쇄하고, 생성된 원소 마나로 18종 스펠을 시전하여 적의 원소 약점을 공략한다. 5개 지역(화산/심해/숲/하늘/심연) × 3층 = 15 스테이지 + 히든 스테이지, 지역 보스 5체 + 최종 보스 + 히든 보스 "원소의 근원" = 7체. 웨이브 간 3택 로그라이크 유물(common 6/rare 4/epic 3), 영구 업그레이드 3트리(공격/방어/유틸), SeededRNG 보석 배치, 3단 난이도(수습/마법사/대마법사), DDA 동적 밸런스, 다국어(한/영). **puzzle+strategy 조합을 1→2개로 강화**한다.

> **MVP 경계**: Phase 1(핵심 루프: 매치-3 전투 → 스펠 시전 → 콤보 체인 → 적 처치 → 보상, 지역 1~2 + 보스 2체 + 업그레이드 기본 트리 + 유물 6종) + Phase 2(지역 3~5 + 보스 3체 + 히든 보스 + 전체 내러티브 + 도전 모드 + 유물 7종) 순차 구현. Phase 1만으로도 완전한 게임 경험 제공 필수.

## §1. 게임 개요 및 핵심 재미 요소

### 1.1 컨셉
엘리멘탈 캐스케이드는 6원소 보석을 매칭하여 마나를 생성하고, 축적된 마나로 강력한 스펠을 시전하여 봉인된 던전의 몬스터를 처치하는 매치-3 로그라이크 RPG다. 매치-3의 **퍼즐 쾌감**, RPG의 **성장 전략**, 로그라이크의 **리플레이 가치**가 삼중 축을 이룬다.

### 1.2 핵심 재미 3축
1. **캐스케이드 콤보**: 보석 스왑 1회가 연쇄 매치(캐스케이드)를 일으키며 화면 가득 폭발하는 시각적/전략적 쾌감. 4매치(십자 폭발), 5매치(원소 폭발), L/T매치(행/열 소거) 등 특수 매치 패턴을 의도적으로 만드는 퍼즐 깊이.
2. **원소 전략**: 6원소 상성 순환(화→대→풍→수→화, 빛↔암) + 보스별 원소 약점 공략 + 스펠 조합. "이번 전투는 물결+바람 마나를 모아 해일 스펠로 갈까, 빛 마나를 축적해서 정화 스펠로 갈까?"
3. **로그라이크 빌드**: 전투 간 3택 유물(common 6/rare 4/epic 3)로 매 런마다 다른 빌드. "콤보 특화 빌드", "단일 원소 집중 빌드", "스펠 쿨다운 빌드" 등 다중 전략 경로.

### 1.3 스토리/내러티브
- **배경**: "캐스케이더"는 원소의 흐름을 조율하여 세계의 균형을 유지하는 마법사 길드다. 고대 봉인이 풀리며 5개 지역의 원소가 폭주하기 시작했다. 마지막 캐스케이더인 플레이어는 각 지역의 원소 균열을 봉인하고, 폭주의 근원을 찾아야 한다.
- **지역별 이야기**: 각 지역에는 봉인을 지키던 수호자의 기록이 남아있다. 화산의 대장장이 이그니스, 심해의 현자 아쿠아, 숲의 정령왕 테라, 하늘의 풍백 에어로, 심연의 이전 캐스케이더 녹스. 전투 사이 짧은 텍스트 대화(Canvas 연출 3줄)로 이야기 전달.
- **엔딩**: 5지역 봉인 완료 → "원소의 안정" 일반 엔딩. 히든 보스 "원소의 근원(Primordial)" 격파 → "캐스케이더의 계승" 진엔딩 — 원소 폭주는 사실 세계가 새로운 균형을 찾기 위한 자연 현상이었고, 캐스케이더의 역할은 억압이 아닌 조율이었다는 반전.
- **스토리 전달**: 전투 간 수호자 기록(3줄 텍스트 + 배경 연출), 보스 등장/격파 컷신(Canvas 카메라 줌 + 대사 5초), 히든 보스 후 에필로그(스크롤 텍스트 10초).

### 1.4 장르 강화 효과
- **puzzle + strategy = 1→2개** (최소 빈도 조합 강화)
- puzzle 전체: 8→9개, strategy 전체: 10→11개
- 기존 puzzle+strategy(runeforge-tactics: 턴제 룬 전략)와 완전 차별화: 매치-3 퍼즐 + RPG 전투 전략 — 완전히 다른 핵심 메카닉

## §2. 게임 규칙 및 목표

### 2.1 기본 규칙
- **그리드 기반 퍼즐**: 8×8 보석 그리드에서 인접 보석을 스왑하여 3개 이상 동일 원소를 일렬로 매칭
- **매치 → 마나 생성**: 매치된 보석의 원소에 해당하는 마나가 축적 (3매치=3마나, 4매치=5마나, 5매치=8마나)
- **마나 → 스펠 시전**: 축적된 마나로 공격/방어/유틸 스펠을 시전하여 적을 공격
- **콤보 배율**: 연쇄 매치(캐스케이드) 횟수 × 0.5 배율 추가 (2콤보=1.5×, 3콤보=2.0×, ...)
- **전투 턴제**: 매치 후 → 스펠 시전 → 적 공격 → 다음 턴. 적 공격 시 플레이어 HP 감소
- **플레이어 HP 0 → 게임 오버**. 획득한 영구 자원(원소 결정)은 보존

### 2.2 매치 시스템 상세

| 매치 유형 | 조건 | 효과 | 마나 생성 |
|----------|------|------|----------|
| 3매치 (기본) | 동일 원소 3개 일렬 | 해당 보석 제거 | 3 |
| 4매치 (십자) | 동일 원소 4개 일렬 | 제거 + 십자 방향 1줄 추가 폭발 | 5 |
| 5매치 (원소 폭발) | 동일 원소 5개 일렬 | 그리드 내 해당 원소 전체 제거 | 8 |
| L매치 | 동일 원소 L자 (3+3, 모서리 공유) | 제거 + 3×3 범위 폭발 | 6 |
| T매치 | 동일 원소 T자 (3+3, 중앙 공유) | 제거 + 행+열 폭발 | 7 |

> 매치 검출 순서: 5매치 → T매치 → L매치 → 4매치 → 3매치 (큰 매치 우선)

### 2.3 지역 구성

| 지역 | 층 수 | 테마 | 환경 효과 | 보스 |
|------|-------|------|----------|------|
| 1. 화산 지대 | 3+보스 | 용암, 화염 기둥, 열기파 | 화염 보석 매치 시 추가 데미지 +20% | 이그니스 (용암 거인) |
| 2. 심해 유적 | 3+보스 | 산호, 해류, 심해 빛 | 물결 보석이 매 턴 1개 랜덤 생성 | 아쿠아 (해연 크라켄) |
| 3. 태고의 숲 | 3+보스 | 거목, 이끼, 포자 | 대지 매치 시 방어력 +15% 버프(2턴) | 테라 (고대 트레인트) |
| 4. 뇌운 하늘 | 3+보스 | 구름, 번개, 극광 | 4매치 이상 시 랜덤 적에게 번개(고정 50 데미지) | 에어로 (폭풍 피닉스) |
| 5. 원소 심연 | 3+보스 | 혼돈, 왜곡, 소멸 | 보석 원소가 매 턴 10% 확률로 무작위 변환 | 녹스 (심연 군주) |
| 히든. 원초의 방 | 보스만 | 전 지역 혼합 | 모든 환경 효과 무작위 순환 (3턴 주기) | 원소의 근원 |

### 2.4 난이도 3종

| 난이도 | 플레이어HP | 적 HP배율 | 턴당 적 공격배율 | 유물 선택지 | 잠금 조건 |
|--------|----------|----------|---------------|-----------|----------|
| 수습 (쉬움) | 150 | ×0.7 | ×0.7 | 4택 (1레어 보장) | 없음 |
| 마법사 (보통) | 100 | ×1.0 | ×1.0 | 3택 | 없음 |
| 대마법사 (어려움) | 70 | ×1.4 | ×1.3 | 3택 (커먼만) | 지역 3 이상 클리어 |

### 2.5 승리/패배 조건
- **스테이지 승리**: 해당 층의 모든 적 처치
- **런 승리**: 5개 지역 순차 클리어 (총 15 일반 스테이지 + 5 보스전 = 20전투)
- **히든 보스**: 3가지 조건 달성 시 16번째 스테이지 "원초의 방" 해금
  - HP 잔여 50% 이상 유지하며 클리어
  - 6가지 원소 스펠 모두 시전한 적 있음
  - 영구 업그레이드 "원소 감지" 해금 완료
- **패배**: HP 0 → 게임 오버 화면 → 원소 결정 정산 → 타이틀

- **유물 선택**: 3개 카드 중 하나를 클릭하여 선택

특수 보석:
- **4매치 생성**: 보석 + 십자 마크 (흰색 테두리 깜빡임)

## §5. 핵심 게임 루프

### 5.1 초기화 순서 (F12 — TDZ 방지)
```
1. CONFIG 상수 선언
2. 전역 상태 객체(G) 선언
3. Canvas DOM 할당 (canvas, ctx)
4. 유틸리티 클래스 초기화 (TweenManager, ObjectPool, SoundManager, SeededRNG)
5. 이벤트 리스너 등록 (키보드, 마우스, 터치, 리사이즈)
6. 게임 루프 시작 (requestAnimationFrame)
```

### 5.2 메인 게임 루프 (60fps)
```
function gameLoop(timestamp) {
  const dt = Math.min(timestamp - lastTime, 33.33); // 최대 2프레임 스킵
  lastTime = timestamp;

  // 1. 입력 처리 (inputManager.process — §6.2 Input 모드 기반)
  // 2. TweenManager 업데이트 (F4: setTimeout 0건, F13: clearImmediate)
  // 3. 상태별 업데이트 — §6.2 ACTIVE_SYSTEMS 매트릭스 참조
  //    - MATCH_IDLE: 보석 스왑 대기, 힌트 타이머
  //    - MATCH_ANIM: 보석 스왑 애니메이션 (tween)
  //    - MATCH_CHECK: 매치 검출 → 제거 → 캐스케이드
  //    - SPELL_SELECT: 스펠 선택 대기
  //    - SPELL_ANIM: 스펠 이펙트 애니메이션
  //    - ENEMY_TURN: 적 공격 애니메이션
  //    - BOSS: 보스 전투 (특수 보석 패턴 + 페이즈 전환)
  // 4. 렌더링 (배경 → 그리드 → 보석 → 적 → 보스 → 스펠이펙트 → 파티클 → UI)
  // 5. ObjectPool 정리 (비활성 파티클 반환)

  requestAnimationFrame(gameLoop);
}
```

> **가드 플래그 패턴 (F5)**: 콤보 정산, 보스 격파 등 상태 전환 tween에는 반드시 `isTransitioning = true` 가드를 설정하여 콜백이 1회만 실행되도록 보호한다.

> **단일 갱신 경로 (F14)**: 모든 애니메이션 값(alpha, scale, position 등)은 TweenManager를 통해서만 갱신한다. 직접 대입과 tween을 동시에 사용하지 않는다.

> **SeededRNG 일관성 (F64)**: 보석 생성, 유물 선택지, 적 행동 등 모든 랜덤은 `G.rng.next()` 경유. `Math.random` 문자열이 코드에 존재하면 스모크 테스트 FAIL.

## §7. 핵심 메카닉 상세

### 7.1 매치-3 엔진

#### 7.1.1 보석 그리드
- 8×8 그리드, 6종 원소 보석 + 2종 특수 보석(4매치 생성, 5매치 생성)
- 초기 보드 생성 시 3매치가 존재하지 않도록 SeededRNG 기반 배치
- 보석 제거 후 상단에서 새 보석이 중력 낙하 (tween 애니메이션)

#### 7.1.2 매치 검출 알고리즘
```
1. 보석 스왑 실행
2. 전체 그리드 스캔:
   - 가로/세로 3+ 연속 동일 원소 검출
   - 5매치 → T매치 → L매치 → 4매치 → 3매치 순으로 우선 판정
3. 매치된 보석 제거 + 마나 생성 + 콤보 카운터 +1
4. 중력 낙하: 빈 셀 위의 보석이 아래로 이동 (tween)
5. 새 보석 생성 (SeededRNG): 상단 빈 셀에 랜덤 원소 보석 생성
6. 다시 2번으로 → 매치 없으면 캐스케이드 종료
7. 콤보 결과 표시 + 마나 정산
```

#### 7.1.3 특수 보석 생성
- **4매치 → 십자 보석**: 해당 원소의 십자 마크 보석 생성. 매치 시 십자 방향 1줄 추가 폭발
- **5매치 → 무지개 보석**: 무지개 색 순환 보석 생성. 다른 보석과 스왑 시 해당 원소 전체 제거
- **L/T매치**: 즉시 범위 폭발 (보석 생성 없음)

#### 7.1.4 힌트 시스템
- 플레이어가 5초간 입력 없으면 가능한 매치 1개를 하이라이트 (보석 반짝임 3초)
- H키 또는 💡 버튼으로 수동 힌트 요청
- 가능한 매치가 없으면 자동 보드 셔플 (SeededRNG)

### 7.2 스펠 시스템

6원소 × 3등급 = 18종 스펠:

| 원소 | 기본 스펠 (마나 3) | 중급 스펠 (마나 6) | 고급 스펠 (마나 10) |
|------|-------------------|-------------------|-------------------|
| 화염 | 파이어볼: 단일 120 DMG | 화염 폭풍: 전체 80 DMG | 메테오: 전체 200 DMG + DoT 3턴 |
| 물결 | 워터 볼트: 단일 100 DMG + 치유 30 | 해일: 전체 70 DMG + 감속 2턴 | 대홍수: 전체 180 DMG + 그리드 물결 보석 +3 |
| 대지 | 록 실드: 방어 +50% (2턴) | 지진: 전체 90 DMG + 스턴 1턴 | 대지의 분노: 전체 220 DMG + 방어파괴 |
| 바람 | 윈드 슬래시: 단일 110 DMG + 회피 +20% | 토네이도: 전체 75 DMG + 보석 셔플 | 태풍의 눈: 전체 190 DMG + 2턴 무적 |
| 빛 | 홀리 레이: 단일 130 DMG (암흑적 ×2) | 정화: 전체 디버프 해제 + 치유 80 | 성스러운 빛: 전체 250 DMG (암흑 ×2) + 치유 100 |
| 암흑 | 섀도 스트라이크: 단일 140 DMG (빛적 ×2) | 저주: 전체 DoT 5턴 (매턴 30) | 심연의 포옹: 전체 200 DMG + 적 공격력 -50% 3턴 |

> 스펠 해금: 기본 스펠은 초기부터 사용 가능. 중급은 영구 업그레이드 "스펠 연구 Lv2"에서 해금. 고급은 "스펠 연구 Lv3"에서 해금.

### 7.3 원소 상성

```
화염 → 대지 (×1.5) → 바람 (×1.5) → 물결 (×1.5) → 화염 (순환)
빛 ↔ 암흑 (상호 ×2.0)
```

- 상성 유리: 데미지 ×1.5
- 상성 불리: 데미지 ×0.7
- 중립: 데미지 ×1.0
- 빛/암흑 상호: 데미지 ×2.0

|---------|---------|--------|------|
| 졸개 (Grunt) | ×1.0 | 10 | 없음 |

### 7.5 보스 전투 상세

#### 이그니스 (용암 거인) — 지역 1 보스
```
Phase 1 (HP 100%~60%): 기본 공격 25 DMG + 화염 보석 3개를 용암 보석으로 변환 (매치 불가, 3턴 후 복원)
Phase 2 (HP 60%~30%): 용암 분출 — 전체 40 DMG + 보드 하단 2줄 용암 보석 변환
Phase 3 (HP 30%~0%): 핵심 노출 — 약점: 물결 스펠 ×2.0 추가. 매턴 20 자가치유
약점 패턴: 물결 보석 5매치 시 약점 노출 (2턴간 받는 데미지 ×3.0)
```

#### 아쿠아 (해연 크라켄) — 지역 2 보스
```
Phase 1 (촉수 4/4): 촉수별 HP 200, 매턴 2촉수 공격 (각 15 DMG)
Phase 2 (촉수 0/4): 본체 노출. 먹물 — 보드 50% 암흑화 (5턴 가려짐)
Phase 3 (HP 40%~0%): 심해 소용돌이 — 매턴 보석 위치 무작위 셔플
약점 패턴: 바람 보석 4매치 이상으로 먹물 해제 + 약점 노출
```

#### 테라 (고대 트레인트) — 지역 3 보스
```
Phase 1 (HP 100%~50%): 뿌리 공격 20 DMG + 보드에 뿌리 보석 4개 (장애물, 파괴 가능)
Phase 2 (HP 50%~0%): 재생 — 매턴 HP 5% 회복. 장애물 6개로 증가
약점 패턴: 화염 보석 매치로 뿌리 장애물 즉시 파괴 + 추가 데미지
```

#### 에어로 (폭풍 피닉스) — 지역 4 보스
```
Phase 1 (HP 100%~60%): 회오리 — 보드 보석 3개 위치 랜덤 스왑 (매턴)
Phase 2 (HP 60%~20%): 폭풍 장벽 — 받는 데미지 50% 감소, 반사 30%
Phase 3 (HP 20%~0%): 부활 준비 — 3턴 내 처치 못하면 HP 30% 회복
약점 패턴: 대지 보석 T매치/L매치로 폭풍 장벽 해제 (2턴간)
```

#### 녹스 (심연 군주) — 지역 5 보스
```
Phase 1 (HP 100%~70%): 원소 변환 — 자신의 원소를 매 2턴 변경 (상성 계속 바뀜)
Phase 2 (HP 70%~40%): 보이드 침식 — 보드 가장자리 보석 6개 암흑화 (매턴)
Phase 3 (HP 40%~0%): 차원 왜곡 — 보석 매치 시 20% 확률로 마나가 잘못된 원소에 축적
약점 패턴: 빛 보석 5매치 "정화" 효과로 모든 암흑화 해제 + 2턴간 원소 고정
```

#### 원소의 근원 (히든 보스)
```
Phase 1 (HP 100%~75%): 6원소 순환 — 매턴 원소 변경, 해당 원소 보석 전부 제거
Phase 2 (HP 75%~50%): 원소 폭주 — 2가지 원소 동시 사용. 보드에 해당 원소 보석만 남김
Phase 3 (HP 50%~25%): 근원의 힘 — 스펠 마나 비용 ×2. 콤보 2 이상만 마나 생성
Phase 4 (HP 25%~0%): 원소 조화 — 6원소 동시 매치(무지개 보석) 시에만 데미지 가능
약점 패턴: 모든 6원소 마나를 동시에 3 이상 보유 시 "조화의 일격" 발동 (전 HP의 10% 고정 데미지)
```

### 8.2 DPS/EHP 밸런스 산식

```
■ 플레이어 DPS (턴당 기대 데미지):
  baseDPS = avgMatchDmg × avgComboMultiplier + spellDPS
  avgMatchDmg = avgMatchCount × elementDmg × matchBonus
  - avgMatchCount ≈ 1.8 (3매치 기준, 통계)
  - elementDmg = CONFIG.BASE_MATCH_DMG × elementMultiplier
  - matchBonus = 1.0 (3매치) / 1.5 (4매치) / 2.5 (5매치)
  avgComboMultiplier ≈ 1.3 (평균 1.6콤보)
  spellDPS = spellDmg / spellCooldown (쿨다운 = 마나 축적 턴 수)

  수습: baseDPS × 1.0 (유물 보너스 가정 +30% → 최종 ≈ baseDPS × 1.3)
  마법사: baseDPS × 1.0 (유물 보너스 가정 +20% → 최종 ≈ baseDPS × 1.2)
  대마법사: baseDPS × 1.0 (유물 커먼만 → 최종 ≈ baseDPS × 1.1)

■ 적 EHP (유효 체력):
  EHP = HP / (1 - damageReduction)
  - damageReduction = 방어/방어율 (중장갑 50%, 나머지 0%)
  - 보스 EHP = 보스HP / (1 - phaseReduction)

■ 클리어 가능성 검증:
  expectedTurns = totalEnemyEHP / playerDPS
  expectedTurns < turnLimit × 0.8 이면 밸런스 OK
  expectedTurns > turnLimit × 0.9 이면 DDA 발동

■ DDA 폴백 (가정 오류 시):
  - 3턴 연속 콤보 0 → 힌트 자동 활성화 + 보석 배치 유리하게 조정
  - HP 30% 이하 + 보스 HP 50% 이상 → 보스 공격력 -20%
  - 5턴 연속 무피해 → 적 HP -5% (숨겨진 원소 침식 연출)

■ 가정:
  - 플레이어 평균 매치 성공률: 수습 95%, 마법사 80%, 대마법사 65%
  - 평균 콤보 횟수: 수습 1.8, 마법사 1.6, 대마법사 1.3
  - 상성 적중률: 60% (올바른 원소 스펠 사용 비율)
  - ⚠️ 가정이 틀릴 경우 DDA 폴백이 3단계로 보정
```

> **유물 캡 (F62)**: DPS 배율 캡 200%, 시너지 캡 150%. 에픽 유물 3연속 선택 시에도 최종 DPS = baseDPS × 2.0을 초과하지 않음.
